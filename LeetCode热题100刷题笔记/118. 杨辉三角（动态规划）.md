给定一个非负整数 numRows，生成「杨辉三角」的前 numRows 行。

在「杨辉三角」中，每个数是它左上方和右上方的数的和。

---
掌握了动态规划的思想之后，这道题的想法就很简单了。根据题意，设结果数组为`ans`，那么对于`ans[i][j]`，应该有`ans[i][j] = ans[i-1][j-1] + ans[i-1][j]`。当然上面的式子在`j==0`或者`j==i`时会涉及一些边界情况，导致上面的式子无定义，但是我们能注意到，这两个位置上的元素必然取值为`1`，所以只需要增加一条额外判断就可以了。
很容易写出如下的代码：
```cpp
class Solution {
public:
    vector<vector<int>> generate(int numRows) {
        vector<vector<int>> ret(numRows);
        for (int i = 0; i < numRows; i++) {
            ret[i].resize(i + 1);
            for (int j = 0; j <= i; j++) {
                if (j == 0 || j == i)
                    ret[i][j] = 1;
                else
                    ret[i][j] = ret[i - 1][j] + ret[i - 1][j - 1];
            }
        }
        return ret;
    }
};
```
[官方题解](https://leetcode.cn/problems/pascals-triangle/solutions/510638/yang-hui-san-jiao-by-leetcode-solution-lew9/)的做法与上面大同小异：
```cpp
class Solution {
public:
    vector<vector<int>> generate(int numRows) {
        vector<vector<int>> ret(numRows);
        for (int i = 0; i < numRows; ++i) {
            ret[i].resize(i + 1);
            ret[i][0] = ret[i][i] = 1;
            for (int j = 1; j < i; ++j) {
                ret[i][j] = ret[i - 1][j] + ret[i - 1][j - 1];
            }
        }
        return ret;
    }
};
```
本题的时间复杂度为$O(numRows^2)$，空间复杂度为$O(1)$（一般不考虑输出值的空间占用）。
