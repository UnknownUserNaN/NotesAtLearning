二叉树中的 路径 被定义为一条节点序列，序列中每对相邻节点之间都存在一条边。同一个节点在一条路径序列中 至多出现一次 。该路径 至少包含一个 节点，且不一定经过根节点。

路径和 是路径中各节点值的总和。

给你一个二叉树的根节点 root ，返回其 最大路径和 。

---
这个问题容易让人联想到前缀和算法，这个算法可以让我们在$O(1)$的时间里计算出数组某一段的部分和。移到二叉树上，我们只需要在遍历二叉树时，记录遍历到当前结点时的前缀和即可。这样，当我们遍历至结点p，想求从其某一个祖先x出发的路径上的元素和的时候，只需要用结点p处的前缀和减去祖先x处的前缀和即可。而对于任意的两个结点p和q，如果要求它们构成的路径上的元素和，也只需要先找到它们的最深的公共祖先x，然后分别求x到p的路径和和x到q的路径和，最后将两者相加（实际上还要再加上结点x的元素值）即可。
现在回到问题本身，我们要求整个二叉树中的最大路径和，采用最常见的递归思路的话，我们可以求每个**在以结点x为根的子树中经过结点x**的最大路径和，然后遍历整个树对应结点x即可，遍历过程中记录这个最大路径和的最大值即可。为什么要规定“**在以结点x为根的子树中经过结点x**”呢？因为这样的设计一方面可以方便我们进行递归，另一方面也方便我们应用前缀和算法。
现在我们来考察如何求取**以结点x为根的子树中经过结点x的最大路径和**。根据前缀和算法，这个路径和可以拆分成两部分来求取，分别是其左子树中的结点p处的前缀和减去结点x处的前缀和，和其右子树中结点q处的前缀和减去结点x处的前缀和。为了令**以结点x为根的子树中经过结点x的路径和**最大， 我们要设法使上面两个分项都取最大值，那这样结点p和q就必须分别是结点x的左右子树中的前缀和最大的结点。
到这里，整个题的思路就已经比较明朗了。首先，为了求整个二叉树中的最大路径和，我们转而遍历二叉树中的每个结点，求取在以当前结点为根且经过该结点的根的最大路径和，并在遍历中记录该路径和的最大值。而为了求得该最大路径和，则需要分别求出遍历至此，其左右两个子树中的前缀和最大的两个结点。而不要忘了，我们还需要遍历一遍二叉树，求出其遍历至此的前缀和。
如果要原原本本地完成上面的思路，那我们实际要按顺序完成如下的几个步骤：
1. 遍历整个二叉树，求出每个结点处的前缀和；
2. 遍历整个二叉树，求出每个结点的左右子树中的前缀和最大值；
3. 遍历整个二叉树，求出以该节点为根的子树中经过该结点的最大路径和；
4. 遍历完成后即可得到整个二叉树的最大路径和。
这需要遍历整个二叉树三次，即时间复杂度为$O(3n)$，而且还需要额外的空间来储存前缀和和子树前缀和最大值等信息。有没有什么办法可以将这三次遍历合为一次呢？显然是可以的。注意到进行步骤1时的深度遍历函数可以不返回值（为void型），我们可以设法令其返回一些有用的东西——比如遍历至此时的**以当前结点为子树的前缀和最大值**。借用这个返回值，我们可以将第2步和第3步都合并到第1步里去。
可以写出下面的程序：
```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left),
 * right(right) {}
 * };
 */
class Solution {
public:
    int ans = INT_MIN;

    int trace(TreeNode* root, int prev_sum) {
        if (root == nullptr)
            return prev_sum;

        // 求遍历至此的前缀和
        int cur_sum = prev_sum + root->val;

        // 求当前结点的左右子树中的前缀和最大值
        // 设置好该回传函数的值，使之返回当前子树中的前缀和最大值
        int left_max_sum = trace(root->left, cur_sum);
        int right_max_sum = trace(root->right, cur_sum);
        // 若子树中的前缀和反而比根节点的要小，那么就相应子树的最大前缀和为当前根节点的值
        left_max_sum = max(cur_sum, left_max_sum);
        right_max_sum = max(cur_sum, right_max_sum);

        // 求取在以该结点为根的子树中经过该结点的最大路径和，并维护答案
        int max_path_sum =
            left_max_sum - cur_sum + right_max_sum - cur_sum + root->val;
        ans = max(ans, max_path_sum);

        // 返回以当前结点为子树的最大的前缀和的值
        return max(max(left_max_sum, right_max_sum), cur_sum);
    }

    int maxPathSum(TreeNode* root) {
        trace(root, 0);
        return ans;
    }
};
```
这样，就写出了时间复杂度为$O(n)$，空间复杂度约为$O(\log n)$的程序。
根据上面的程序，其实可以抽象出递归方法的一般模板：
```text
返回值类型 递归函数(当前位置的参数, 表示累计的参数){
	if(当前位置已经到达尽头/递归的子问题规模已经最小)
		return 递归返回值;

	前处理逻辑
	递归后的结果 = 递归函数(下一位置的参数, 表示累计的参数);
	后处理逻辑

	return 递归返回值;
}
```
[官方题解](https://leetcode.cn/problems/binary-tree-maximum-path-sum/solutions/297005/er-cha-shu-zhong-de-zui-da-lu-jing-he-by-leetcode-/)的做法更为简洁，将前面的前缀和算法“更新”成了“增益”这个概念。对于遍历过程中的每一个结点，我们考虑从该结点出发的所有路径的路径和，以及它们所对应的“增益”。
> 首先，考虑实现一个简化的函数 maxGain(node)，该函数计算二叉树中的一个节点的最大贡献值，具体而言，就是在以该节点为根节点的子树中寻找以该节点为起点的一条路径，使得该路径上的节点值之和最大。
> 具体而言，该函数的计算如下。
> 	空节点的最大贡献值等于 0。
> 	非空节点的最大贡献值等于节点值与其子节点中的最大贡献值之和（对于叶节点而言，最大贡献值等于节点值）。
> 根据函数 maxGain 得到每个节点的最大贡献值之后，如何得到二叉树的最大路径和？对于二叉树中的一个节点，该节点的最大路径和取决于该节点的值与该节点的左右子节点的最大贡献值，如果子节点的最大贡献值为正，则计入该节点的最大路径和，否则不计入该节点的最大路径和。维护一个全局变量 maxSum 存储最大路径和，在递归过程中更新 maxSum 的值，最后得到的 maxSum 的值即为二叉树中的最大路径和。

```cpp
class Solution {
private:
    int maxSum = INT_MIN;

public:
    int maxGain(TreeNode* node) {
        if (node == nullptr) {
            return 0;
        }
        
        // 递归计算左右子节点的最大贡献值
        // 只有在最大贡献值大于 0 时，才会选取对应子节点
        int leftGain = max(maxGain(node->left), 0);
        int rightGain = max(maxGain(node->right), 0);

        // 节点的最大路径和取决于该节点的值与该节点的左右子节点的最大贡献值
        int priceNewpath = node->val + leftGain + rightGain;

        // 更新答案
        maxSum = max(maxSum, priceNewpath);

        // 返回节点的最大贡献值
        return node->val + max(leftGain, rightGain);
    }

    int maxPathSum(TreeNode* root) {
        maxGain(root);
        return maxSum;
    }
};
```
