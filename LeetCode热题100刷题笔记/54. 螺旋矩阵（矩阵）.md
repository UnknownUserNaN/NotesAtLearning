给你一个 m 行 n 列的矩阵 matrix ，请按照 顺时针螺旋顺序 ，返回矩阵中的所有元素。

---
一个非常简单的思路是，按照顺时针螺旋，也就是“右、下、左、上”的顺序走遍整个矩阵。一边走一边输出元素，然后将走过的位置打上标记。走的过程中还要注意下一步到底能不能走，如果不能走则按上面的顺序切换行走方向。如此重复，直至遍历完所有元素。
基于上面的思路，很容易写出下面的代码。当然，这个代码还有足够的改善空间。
```cpp
class Solution {
public:
    vector<int> spiralOrder(vector<vector<int>>& matrix) {
        int m = matrix.size();
        int n = matrix[0].size();
        const int marker = 65989546; // 无规律数字做走过的“标记”

        // 遍历整个数组，一边走一边给相应元素做“标记”
        vector<int> ans;
        int i = 0, j = 0; // 当前位置
        int dir = 0;      // 行进方向：0=→, 1=↓, 2=←, 3=↑
        while (ans.size() < m * n) {
            ans.emplace_back(matrix[i][j]); // 将当前位置加入答案
            matrix[i][j] = marker;          // 使用标记指示当前位置已经走过了

            // 检查当前方向上是否可以行走到下一步
            if (dir == 0) {
                if (j == n - 1 || matrix[i][j + 1] == marker) {
                    dir++;
                    i++;
                } else
                    j++;
            } else if (dir == 1) {
                if (i == m - 1 || matrix[i + 1][j] == marker) {
                    dir++;
                    j--;
                } else
                    i++;
            } else if (dir == 2) {
                if (j == 0 || matrix[i][j - 1] == marker) {
                    dir++;
                    i--;
                } else
                    j--;
            } else if (dir == 3) {
                if (i == 0 || matrix[i - 1][j] == marker) {
                    dir = 0;
                    j++;
                } else
                    i--;
            }
        }

        return ans;
    }
};
```
不过很明显，这样的做法需要修改原来的数组，并且还要使用“标记”这种不“安全”的做法，显然不够合适。
不打标记自然也是可以完成上面的任务的，只不过就需要研究一下螺旋式行进的输出特点。首先，遍历肯定是按照“右、下、左、上”的顺序进行的，那么在这些方向上行进的步数有没有规律呢？我们设想一个m行n列的矩阵，我们会按照如下的方式行进：
1. 向右走n-1步；
2. 向下走m-1步；
3. 向左走n-1步；
4. 向上走m-2步，因为第1步的路径已经使得这一方向减少了一步；
5. 向右走n-2步，因为第2步的路径已经使得这一方向减少了一步；
6. 向左走m-3步，因为第2步和第3步的路径已经使得这一方向减少了一步；
7. 向上走n-3步，因为第3步和4步的路径已经使得这一方向减少了一步；
8. ……
可以看出，这个行进步数组成的数列为`[n-1, m-1, n-1, m-2, n-2, m-3, n-3, ...]`，除了数列的第一项为n-1不符合规律外，其余每两项均可视为一组，每组依次出现且值依次减一，直至有元素为0。
为了使其在一开始就符合规律，我们可以设定初始时的位置在第1行、第“0”列处，这样数列就会变为`[n, m-1, n-1, m-2, n-2, m-3, ...]`，方便处理了。
随后，我们只需要模仿上面的行进规律行进即可。基于此可以写出下面的代码：
```cpp
class Solution {
public:
    vector<int> spiralOrder(vector<vector<int>>& matrix) {
        int m = matrix.size();
        int n = matrix[0].size();

        // 遍历整个数组
        vector<int> ans;
        int i = 0, j = -1; // 当前位置
        int dir = 0;      // 行进方向：0=→, 1=↓, 2=←, 3=↑
        int mn_steps[2] = {n, m-1}; // 步数规律
        while(ans.size()<m*n){
            // 按规律行进
            for(int leave_steps = mn_steps[0]; leave_steps>0; leave_steps--){
                if(dir == 0) j++;
                else if(dir == 2) j--;
                ans.emplace_back(matrix[i][j]);
            }
            dir = (dir+1)%4;
            for(int leave_steps = mn_steps[1]; leave_steps>0; leave_steps--){
                if(dir == 1) i++;
                else if(dir == 3) i--;
                ans.emplace_back(matrix[i][j]);
            }
            dir = (dir+1)%4;

            // 处理目标的步数
            mn_steps[0]--;
            mn_steps[1]--;
        }

        return ans;
    }
};
```
官方题解的思路实际上与上面的做法大同小异，只不过官方使用“层”的概念来解释每一步行进的区别：
```cpp
class Solution {
public:
    vector<int> spiralOrder(vector<vector<int>>& matrix) {
        if (matrix.size() == 0 || matrix[0].size() == 0) {
            return {};
        }

        int rows = matrix.size(), columns = matrix[0].size();
        vector<int> order;
        int left = 0, right = columns - 1, top = 0, bottom = rows - 1;
        while (left <= right && top <= bottom) {
            for (int column = left; column <= right; column++) {
                order.push_back(matrix[top][column]);
            }
            for (int row = top + 1; row <= bottom; row++) {
                order.push_back(matrix[row][right]);
            }
            if (left < right && top < bottom) {
                for (int column = right - 1; column > left; column--) {
                    order.push_back(matrix[bottom][column]);
                }
                for (int row = bottom; row > top; row--) {
                    order.push_back(matrix[row][left]);
                }
            }
            left++;
            right--;
            top++;
            bottom--;
        }
        return order;
    }
};
```
