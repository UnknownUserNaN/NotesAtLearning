以数组 `intervals` 表示若干个区间的集合，其中单个区间为 `intervals[i] = [start_i, end_i]` 。请你合并所有重叠的区间，并返回 *一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间* 。

***

除了最基本的暴力方法，这个题没有想出什么比较好的思路，主要是难以估计这个算法最基本的复杂度。

但看了官方题解之后，实际上，会发现这个题目的做法非常自然。根据题目的示例可以发现，示例中的各个区间都是按左端点从小到大排序的，这有利于我们合并区间。按左端点由小到大排序是一个很好的性质，但本题的测试用例往往不能完全保证这个性质，因此我们首先需要对输入的各个区间进行排序。

排序完成之后，就可以逐个遍历各个元素了。首先将第一个元素放入输出列表中，然后逐个遍历其他元素，如果该元素的左端点在末尾元素的区间内，则该区间可以与最后的区间合并，否则则应单独加入输出列表，单独作为一个区间。

官方题解还用了反证法证明了这种做法的正确性，但我觉得这实际上是十分自然地一种想法。下面给出我自己按照上面的思路编写的程序。

```cpp
class Solution {
public:
    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        // 特殊处理
        if (intervals.empty())
            return {{}};

        // 排序
        sort(intervals.begin(), intervals.end());

        // 逐个遍历元素
        vector<vector<int>> ans;
        ans.push_back(intervals[0]);
        for (int i = 1; i < intervals.size(); i++) {
            // 检查当前遍历元素是否可以合并
            if (intervals[i][0] <= ans.back()[1]) {
                // 可以合并
                ans.back()[0] = min(intervals[i][0], ans.back()[0]);
                ans.back()[1] = max(intervals[i][1], ans.back()[1]);
            } else {
                // 不可以合并
                ans.push_back(intervals[i]);
            }
        }

        return ans;
    }
};
```

有点可惜，本来这道题可以自己独立做出来的，可惜还是没有坚持自己的想法，以为自己想到的想法因为要用到排序，所以不是最简单的做法，但事实恰恰正是如此。这样的算法是`$O(n\log(n))$`的复杂度，已经是最低了。
