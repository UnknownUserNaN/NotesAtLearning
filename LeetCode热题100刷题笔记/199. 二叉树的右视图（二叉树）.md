给定一个二叉树的 根节点 root，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

---
这个问题非常适合使用二叉树的按层次遍历，也就是广度优先的遍历。每遍历完一层，即将当前层的队列的最后一个元素放入结果数组。如此不断重复即可。
```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left),
 * right(right) {}
 * };
 */
class Solution {
public:
    vector<int> rightSideView(TreeNode* root) {
        if (root == nullptr)
            return {};

        TreeNode* p = root;
        vector<int> ans;
        queue<TreeNode*> q;
        q.push(root);
        while (!q.empty()) {
            int n = q.size();
            ans.emplace_back(q.back()->val);
            for (int i = 0; i < n; i++) {
                p = q.front();
                q.pop();

                if (p->left != nullptr)
                    q.push(p->left);
                if (p->right != nullptr)
                    q.push(p->right);
            }
        }
        return ans;
    }
};
```
这个程序的时间复杂度依然为$O(n)$，最差情况下的空间复杂度为$O(n)$。
这个算法也可以使用深度优先的算法完成，只不过需要注意两点，第一个是遍历时优先考虑右子树（保证走到最右侧），第二个是一边遍历一边也要记录深度信息（如果同一深度下先后遍历到了两个结果，那么一定是先被记录的位于最右侧）。
下面是官方题解的实现方法，有点复杂，但其时间复杂度和空间复杂度与之前方法相同：
```cpp
class Solution {
public:
    vector<int> rightSideView(TreeNode* root) {
        unordered_map<int, int> rightmostValueAtDepth;
        int max_depth = -1;

        stack<TreeNode*> nodeStack;
        stack<int> depthStack;
        nodeStack.push(root);
        depthStack.push(0);

        while (!nodeStack.empty()) {
            TreeNode* node = nodeStack.top();nodeStack.pop();
            int depth = depthStack.top();depthStack.pop();

            if (node != NULL) {
            	// 维护二叉树的最大深度
                max_depth = max(max_depth, depth);

                // 如果不存在对应深度的节点我们才插入
                if (rightmostValueAtDepth.find(depth) == rightmostValueAtDepth.end()) {
                    rightmostValueAtDepth[depth] =  node -> val;
                }

                nodeStack.push(node -> left);
                nodeStack.push(node -> right);
                depthStack.push(depth + 1);
                depthStack.push(depth + 1);
            }
        }

        vector<int> rightView;
        for (int depth = 0; depth <= max_depth; ++depth) {
            rightView.push_back(rightmostValueAtDepth[depth]);
        }

        return rightView;
    }
};
```
评论区有人给出了比上面方法更为简洁的实现方法，注释也很详细：
```cpp
class Solution {
public:
    vector<int> rightSideView(TreeNode* root) {
        vector<int> res; // 存储右视图结果
        if (!root) return res; // 处理空树情况
        stack<pair<TreeNode*, int>> stk; // 存储节点和对应深度
        stk.emplace(root, 0); // 根节点深度为0
        while (!stk.empty()) {
            auto [node, depth] = stk.top(); // 获取栈顶节点和深度
            stk.pop();
            // 关键判断：当深度等于res的大小时，说明是该层第一个访问的节点
            if (depth == res.size()) 
                res.push_back(node->val);
            // 注意入栈顺序：先左后右，保证出栈时先右后左
            if (node->left) stk.emplace(node->left, depth + 1);
            if (node->right) stk.emplace(node->right, depth + 1);
        }
        return res;
    }
};
```
