给你二叉树的根结点 root ，请你将它展开为一个单链表：
- 展开后的单链表应该同样使用 TreeNode ，其中 right 子指针指向链表中下一个结点，而左子指针始终为 null 。
- 展开后的单链表应该与二叉树 先序遍历 顺序相同。
进阶：你可以使用原地算法（O(1) 额外空间）展开这棵树吗？

---
如果允许新增数组空间，那么思路就会很简单，先写一个先序遍历的函数，然后将其中遍历到的元素“复制”到新建的链表之中。但是本题要求的函数返回值是`void`，所以这里必然是要求我们采用“原地”算法了。
考虑先序遍历的过程，其输出顺序必然是先输出根，再输出左子树，再输出它的右子树。然后对于其两个子树，则可以递归地考虑上述过程。要将一棵树展开成链表，也可以遵循类似的递归步骤，具体步骤是先将根节点连接到链表上，然后是再处理左子树，然后是右子树。写出的程序如下：

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left),
 * right(right) {}
 * };
 */
class Solution {
    TreeNode* last = nullptr;

public:
    void recursiveFlatten(TreeNode* root) {
        if (last != nullptr) {
            last->left = nullptr;
            last->right = root;
        }
        last = root;

        // 提前存储当前节点的左右孩子，防止因结点状态改变而指错结点
        TreeNode *pl = root->left, *pr = root->right;
        if (pl != nullptr)
            recursiveFlatten(pl);
        if (pr != nullptr)
            recursiveFlatten(pr);
    }

    void flatten(TreeNode* root) {
        if (root != nullptr)
            recursiveFlatten(root);
    }
};
```
上面是采用递归方法实现的，实际上也可以通过迭代法结合栈结构来实现：
```cpp
class Solution {
public:
    void flatten(TreeNode* root) {
        if (root == nullptr)
            return;

        stack<TreeNode*> stk;
        stk.push(root);
        TreeNode* last = nullptr;
        while (!stk.empty()) {
            TreeNode* cur = stk.top();
            stk.pop();

            if (last != nullptr) {
                last->left = nullptr;
                last->right = cur;
            }
            last = cur;

            TreeNode *pl = cur->left, *pr = cur->right;
            // 注意入栈顺序，由于栈是先进后出，所以要让右孩子先入栈才能实现后出
            if (pr != nullptr)
                stk.push(pr);
            if (pl != nullptr)
                stk.push(pl);
        }
    }
};
```
这个算法的时间复杂度为$O(n)$，最坏情况下的空间复杂度为$O(n)$。使用了线性空间的原因是调用了递归隐性使用了不少栈空间。本题指出存在$O(1)$额外复杂度的算法，实际上可以参考之前Morris算法来实现，也就是在每次遍历时找中继节点（predecessor）（相当于创建回传指针）。具体可参考第94题“二叉树的中序遍历”的题解。
下面程序的关键在于，找到中继结点之后，要意识到**中继结点是左子树最后一个遍历到的点。按照遍历顺序，其下一个结点是右子树的根节点**。所以为了满足题目要求，**这里直接将右子树“接”到中继结点处，使其成为中继结点的右孩子。**
```cpp
class Solution {
public:
    void flatten(TreeNode* root) {
        if (root == nullptr)
            return;

        TreeNode *p = root, *pred = nullptr;
        while (p != nullptr) {
            if (p->left == nullptr) {
                // Morris算法这里需要执行输出当前结点结果的操作，但对此题不需要
                // 可以直接跳转到右指针
                p = p->right;
            } else {
                pred = p->left;
                while (pred->right != nullptr && pred->right != p->right)
                    pred = pred->right;

                if (pred->right == nullptr) {
                    // 与Morris算法不同，这里直接令中继节点的右孩子是当前节点p的右孩子，而不是结点p
                    // 这里是直接将p的右子树移到pred的右孩子处
                    pred->right = p->right;
                    p->right = p->left;
                    p->left = nullptr;
                    p = p->right;
                } else {
                    // Morris算法这里要求输出p的值，然后恢复原来的连接关系，但本题都不需要做这些事
                }
            }
        }
    }
};
```

官方题解的程序也写的较为简练：
```cpp
class Solution {
public:
    void flatten(TreeNode* root) {
        TreeNode *curr = root;
        while (curr != nullptr) {
            if (curr->left != nullptr) {
                auto next = curr->left;
                auto predecessor = next;
                while (predecessor->right != nullptr) {
                    predecessor = predecessor->right;
                }
                predecessor->right = curr->right;
                curr->left = nullptr;
                curr->right = next;
            }
            curr = curr->right;
        }
    }
};
```
