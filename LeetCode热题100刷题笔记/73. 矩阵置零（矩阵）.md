给定一个 *`m`*`  x  `*`n`* 的矩阵，如果一个元素为 **0 ，则将其所在行和列的所有元素都设为 0** 。请使用 **[原地](http://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95)** 算法。

**提示：**

*   `m == matrix.length`
*   `n == matrix[0].length`

**进阶：**

*   一个直观的解决方案是使用  `O(`*`mn`*`)` 的额外空间，但这并不是一个好的解决方案。
*   一个简单的改进方案是使用 `O(`*`m`*` + `*`n`*`)` 的额外空间，但这仍然不是最好的解决方案。
*   你能想出一个仅使用常量空间的解决方案吗？

***

首先，本题的时间复杂度最低一定是`$O(mn)$`，也就是必定要遍历一次矩阵。进阶中所谓“`$O(mn)$`“的额外空间，肯定是新建一个与输入矩阵同形的矩阵然后覆盖掉原矩阵，但这显然不够经济。使用”`$O(m+n)$`“的额外空间，指的是记录对应矩阵的各行各列是否需要置零，`$m$`行和`$n$`列，恰好构成`$O(m+n)$`的复杂度。
本题明确说了要采用“原地”算法，所以上面两种比较简单的思路自然是行不通了。根据题目提示，我们可以考虑尝试将第一行和第一列的元素用来作为某行或某列需要置零的标记。首先遍历整个矩阵，如果遇到了0元素，便将相应行列首个元素置为相应元素。然后遍历第一行和第一列的元素，如果遇到了标记则将相应行列的元素（除去第一个元素）置0。最后再次遍历第一行和第一列的元素，将遇到的标记置0即可。
基于上面的思路可以写出下面的代码：
```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int m = matrix.size();
        int n = matrix[0].size(); // 矩阵尺寸为m×n
        const int marker = 5416587; // 采用任意整数作为行列标记

        // 遍历矩阵，并给相应行和列赋予标记
        for(int i=0; i<m; i++){
            for(int j=0; j<n; j++){
                if(matrix[i][j] == 0){
                    matrix[i][0] = marker;
                    matrix[0][j] = marker;
                }
            }
        }

        // 遍历每个行和每个列，若恰有标记则给相应行和相应列赋零
        for(int i=0; i<m; i++){
            if(matrix[i][0] == marker){
                for(int j=0; j<n; j++){
                    if(matrix[i][j] != marker)
                        matrix[i][j] = 0;
                }
            }
        }
        for(int j=0; j<n; j++){
            if(matrix[0][j] == marker){
                for(int i=0; i<m; i++){
                    if(matrix[i][j] != marker)
                        matrix[i][j] = 0;
                }
            }
        }

        // 最后将所有的marker置零
        for(int i=0; i<m; i++){
            if(matrix[i][0] == marker)
                matrix[i][0] = 0;
        }
        for(int j=0; j<n; j++){
            if(matrix[0][j] == marker)
                matrix[0][j] = 0;
        }
    }
};
```
可惜的是，这个程序能通过一部分的用例，在下面这个测试用例上出错了：
```
[[1, 2, 3, 4 ],
 [5, 0, 7, 8 ],
 [0, 10,11,12],
 [13,14,15,0 ]]
```
经检查，问题出在“9”的位置上的那个0。按照上面的程序，它让“1”的位置上打上了标记。这个标记会使得第一行和第一列都被置零，但是我们的原意只有第一列置零（即1、5、0、13这一列）。这本质上是“1”这个位置打标记的歧义性。
好在这个会造成歧义性的位置仅此一个，因此只需要对这一个位置特殊处理就好了。使用一个新的变量记录“1”这个位置究竟是要让所在行置零、还是所在列置零、还是所在的行和列均置零。当然也可以换一种更为“优雅”的处理办法，即采用三种不同的标记，即“行标记”“列标记”和“行列标记”。基于这个思路可以写出下面更正后的代码：
```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int m = matrix.size();
        int n = matrix[0].size(); // 矩阵尺寸为m×n
        const int i_marker = 5416587;
        const int j_marker = 5642894; // 采用任意整数作为行列标记
        const int ij_marker = 96584168;

        // 遍历矩阵，并给相应行和列赋予标记
        for (int i = 0; i < m; i++)
            for (int j = 0; j < n; j++)
                if (matrix[i][j] == 0) {
                    if (matrix[i][0] == j_marker || matrix[i][0] == ij_marker)
                        matrix[i][0] = ij_marker;
                    else
                        matrix[i][0] = i_marker;
                    if (matrix[0][j] == i_marker || matrix[0][j] == ij_marker)
                        matrix[0][j] = ij_marker;
                    else
                        matrix[0][j] = j_marker;
                }

        // 遍历每个行和每个列，若恰有标记则给相应行和相应列赋零
        for (int i = 0; i < m; i++)
            if (matrix[i][0] == i_marker || matrix[i][0] == ij_marker)
                for (int j = 0; j < n; j++)
                    if (matrix[i][j] != i_marker && matrix[i][j] != j_marker &&
                        matrix[i][j] != ij_marker)
                        matrix[i][j] = 0;
        for (int j = 0; j < n; j++)
            if (matrix[0][j] == j_marker || matrix[0][j] == ij_marker)
                for (int i = 0; i < m; i++)
                    if (matrix[i][j] != i_marker && matrix[i][j] != j_marker &&
                        matrix[i][j] != ij_marker)
                        matrix[i][j] = 0;

        // 最后将所有的marker置零
        for (int i = 0; i < m; i++)
            if (matrix[i][0] == i_marker || matrix[i][0] == ij_marker)
                matrix[i][0] = 0;
        for (int j = 0; j < n; j++)
            if (matrix[0][j] == j_marker || matrix[0][j] == ij_marker)
                matrix[0][j] = 0;
    }
};
```
修改后的代码增加了很多条件判断，变得复杂了一些，但总算实现了原地算法，甚至一个额外变量都没有使用。当然，这样的做法显然不够“安全”，因为上面输入的随机的数实际上只能通过有限的测试用例。
官方题解的方法相对更为严谨一些，不过需要使用两个额外变量，记录第一行和第一列是否需要置零。因为采用第一行和第一列来“存放标记”时会“冲刷”掉原来的元素，因此需要使用额外的变量来指示原来的信息。这与我们改进算法的思路不谋而和。
其写出的代码如下：
```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int m = matrix.size();
        int n = matrix[0].size();
        int flag_col0 = false, flag_row0 = false;

		// 检查第一行和第一列中是否有0，有0则记录
        for (int i = 0; i < m; i++) {
            if (!matrix[i][0]) {
                flag_col0 = true;
            }
        }
        for (int j = 0; j < n; j++) {
            if (!matrix[0][j]) {
                flag_row0 = true;
            }
        }

		// 遍历数组中除了第一行和第一列以外的元素，若有0则立即将当前行列置0
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (!matrix[i][j]) {
                    matrix[i][0] = matrix[0][j] = 0;
                }
            }
        }
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (!matrix[i][0] || !matrix[0][j]) {
                    matrix[i][j] = 0;
                }
            }
        }
        
        // 再回过头来对第一行和第一列的元素进行处理
        if (flag_col0) {
            for (int i = 0; i < m; i++) {
                matrix[i][0] = 0;
            }
        }
        if (flag_row0) {
            for (int j = 0; j < n; j++) {
                matrix[0][j] = 0;
            }
        }
    }
};
```
官方题解还有一种“只用一个额外变量”的算法，当然这种就太“走火入魔”了，这里就不赘述了。
