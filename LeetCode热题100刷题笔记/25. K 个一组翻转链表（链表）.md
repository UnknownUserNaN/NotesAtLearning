给你链表的头节点 head ，每 k 个节点一组进行翻转，请你返回修改后的链表。

k 是一个正整数，它的值小于或等于链表的长度。如果节点总数不是 k 的整数倍，那么请将最后剩余的节点保持原有顺序。

你不能只是单纯的改变节点内部的值，而是需要实际进行节点交换。

进阶：你可以设计一个只用 O(1) 额外内存空间的算法解决此问题吗？

---
这题其实是前面24题的“进化版”，但其实思路大同小异。拆解原来的几个步骤即可：
1. 在链表头部增加哑结点，并令操作的指针指向哑结点；
2. 查询当前指针接下来的k个结点是否存在；
3. 若存在，则翻转接下来遇到的k个结点；
4. 移动指针到下一个合适的位置，并回到第2步。
为了方便程序编写不出问题，可以将上面的第2步和第3步都“打包”成函数。
最终写出的程序如下：
```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    bool checkExist(ListNode* p, int k) {
        for (int i = 0; i < k; i++) {
            if (p->next == nullptr)
                return false;
            else
                p = p->next;
        }
        return true;
    }

    ListNode* flipNodes(ListNode* head, int k) {
        ListNode* prev = head;
        ListNode* cur = head->next;
        ListNode* next = head->next->next;
        ListNode* start = head->next;
        for (int i = 0; i < k; i++) {
            if (i == k - 1) {
                start->next = cur->next;
                head->next = cur;
            }
            cur->next = prev;
            prev = cur;
            cur = next;
            next = cur == nullptr ? nullptr : cur->next;
        }
        return start;
    }

    ListNode* reverseKGroup(ListNode* head, int k) {
        if (k == 1)
            return head;
        ListNode* prehead = new ListNode(0, head);
        ListNode* p = prehead;
        while (checkExist(p, k)) {
            p = flipNodes(p, k);
        }
        ListNode* ret = prehead->next;
        delete prehead;
        return ret;
    }
};
```
这个题目有两个边界条件的处理比较烦人：
1. 当k=1时需要另外处理，因为编写的flipNodes函数会在k=1时构成环；
2. 函数flipNodes中，进行最后一次循环时，next的赋值会因为cur变成nullptr而出现困难，需要额外的判断；
这个题其实思路上并不难，主要难在实现上。官方题解的做法也比较冗长，但比着我的写法要更简洁一些：
```cpp
class Solution {
public:
    // 翻转一个子链表，并且返回新的头与尾
    pair<ListNode*, ListNode*> myReverse(ListNode* head, ListNode* tail) {
        ListNode* prev = tail->next;
        ListNode* p = head;
        while (prev != tail) {
            ListNode* nex = p->next;
            p->next = prev;
            prev = p;
            p = nex;
        }
        return {tail, head};
    }

    ListNode* reverseKGroup(ListNode* head, int k) {
        ListNode* hair = new ListNode(0);
        hair->next = head;
        ListNode* pre = hair;

        while (head) {
            ListNode* tail = pre;
            // 查看剩余部分长度是否大于等于 k
            for (int i = 0; i < k; ++i) {
                tail = tail->next;
                if (!tail) {
                    return hair->next;
                }
            }
            ListNode* nex = tail->next;
            // 这里是 C++17 的写法，也可以写成
            // pair<ListNode*, ListNode*> result = myReverse(head, tail);
            // head = result.first;
            // tail = result.second;
            tie(head, tail) = myReverse(head, tail);
            // 把子链表重新接回原链表
            pre->next = head;
            tail->next = nex;
            pre = tail;
            head = tail->next;
        }

        return hair->next;
    }
};
```
官方题解中，“子链表”的抽象很巧妙，尤其是在翻转k个结点时传入首结点和尾结点，在很大程度上简化了程序的设计，也免去了许多边界条件的处理。
