给你链表的头结点 head ，请将其按 升序 排列并返回 排序后的链表 。

进阶：你可以在 O(n log n) 时间复杂度和常数级空间复杂度下，对链表进行排序吗？

---
由于链表的结构，其天然适合插入和删除，因此采用插入法排序链表写起来会非常简单：
```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* sortList(ListNode* head) {
        // 增加头结点
        ListNode *prehead_old = new ListNode(0, head),
                 *prehead_new = new ListNode();

        // 使用插入法排序
        ListNode *p1 = prehead_old, *p2 = prehead_new;
        while (prehead_old->next != nullptr) {
            p1 = prehead_old;
            ListNode* min_node = prehead_old;
            int min_val = INT_MAX;
            while (p1->next != nullptr) {
                if (p1->next->val <= min_val) {
                    min_val = p1->next->val;
                    min_node = p1;
                }
                p1 = p1->next;
            }
            p2->next = min_node->next;
            min_node->next = min_node->next->next;
            p2 = p2->next;
        }

        ListNode* ret = prehead_new->next;
        delete prehead_old, prehead_new;
        return ret;
    }
};
```
但是，插入法的时间复杂度是$O(n^2)$，其在长链表排序时非常容易超时。实际上，本题的”进阶“就已经提示了我们，本题实际上存在跟经典的快排一样的$O(n\log n)$时间的解法。
[官方题解](https://leetcode.cn/problems/sort-list/solutions/492301/pai-xu-lian-biao-by-leetcode-solution/)中指出：
> 题目的进阶问题要求达到 $O(n\log n)$ 的时间复杂度和 $O(1)$ 的空间复杂度，时间复杂度是 $O(n\log n)$ 的排序算法包括**归并排序、堆排序和快速排序**（快速排序的最差时间复杂度是 $O(n^2）$，其中最适合链表的排序算法是归并排序。
> 归并排序基于**分治算法**。最容易想到的实现方式是自顶向下的递归实现，考虑到递归调用的栈空间，自顶向下归并排序的空间复杂度是$O(\log n)$。如果要达到 $O(1)$ 的空间复杂度，则需要使用自底向上的实现方式。

归并排序有两种，一种是自顶向下的：
> 对链表自顶向下归并排序的过程如下。
> 1. 找到链表的中点，以中点为分界，将链表拆分成两个子链表。寻找链表的中点可以使用快慢指针的做法，快指针每次移动 2 步，慢指针每次移动 1 步，当快指针到达链表末尾时，慢指针指向的链表节点即为链表的中点。
> 2. 对两个子链表分别排序。
> 3. 将两个排序后的子链表合并，得到完整的排序后的链表。（可以使用「21. 合并两个有序链表」的做法，将两个有序的子链表进行合并。）
> 上述过程可以通过递归实现。递归的终止条件是链表的节点个数小于或等于 1，即当链表为空或者链表只包含 1 个节点时，不需要对链表进行拆分和排序。

程序如下：
```cpp
class Solution {
public:
    ListNode* sortList(ListNode* head) {
        return sortList(head, nullptr);
    }

    ListNode* sortList(ListNode* head, ListNode* tail) {
	    // 采用递归方法排序链表
        if (head == nullptr) {
            return head;
        }
        if (head->next == tail) {
            head->next = nullptr;
            return head;
        }

		// 下面是使用快慢指针法找到链表的中点，并依此分割链表
        ListNode* slow = head, *fast = head;
        while (fast != tail) {
            slow = slow->next;
            fast = fast->next;
            if (fast != tail) {
                fast = fast->next;
            }
        }
        ListNode* mid = slow;

		// 用递归方法排序链表
		// 将链表不断二分，直到拆分成单个元素，使用merge函数便可自动完成“排序”的过程
        return merge(sortList(head, mid), sortList(mid, tail)); 
    }

    ListNode* merge(ListNode* head1, ListNode* head2) {
	    // 合并两个有序数组（参见21题）
        ListNode* dummyHead = new ListNode(0);
        ListNode* temp = dummyHead, *temp1 = head1, *temp2 = head2;
        while (temp1 != nullptr && temp2 != nullptr) {
            if (temp1->val <= temp2->val) {
                temp->next = temp1;
                temp1 = temp1->next;
            } else {
                temp->next = temp2;
                temp2 = temp2->next;
            }
            temp = temp->next;
        }
        if (temp1 != nullptr) {
            temp->next = temp1;
        } else if (temp2 != nullptr) {
            temp->next = temp2;
        }
        return dummyHead->next;
    }
};
```
这种方法由于使用了递归，因此空间复杂度较高，达到了$O(\log n)$。
另一种归并排序是“自底向上”的，其方法如下：
> 首先求得链表的长度 length，然后将链表拆分成子链表进行合并。
> 具体做法如下。
> 1. 用 subLength 表示每次需要排序的子链表的长度，初始时 subLength=1。
> 2. 每次将链表拆分成若干个长度为 subLength 的子链表（最后一个子链表的长度可以小于 subLength），按照每两个子链表一组进行合并，合并后即可得到若干个长度为 subLength×2 的有序子链表（最后一个子链表的长度可以小于 subLength×2）。合并两个子链表仍然使用「21. 合并两个有序链表」的做法。
> 3. 将 subLength 的值加倍，重复第 2 步，对更长的有序子链表进行合并操作，直到有序子链表的长度大于或等于 length，整个链表排序完毕。
> 如何保证每次合并之后得到的子链表都是有序的呢？可以通过数学归纳法证明。
> 1. 初始时 subLength=1，每个长度为 1 的子链表都是有序的。
> 2. 如果每个长度为 subLength 的子链表已经有序，合并两个长度为 subLength 的有序子链表，得到长度为 subLength×2 的子链表，一定也是有序的。
> 3. 当最后一个子链表的长度小于 subLength 时，该子链表也是有序的，合并两个有序子链表之后得到的子链表一定也是有序的。
> 因此可以保证最后得到的链表是有序的。

写出的程序如下：
```cpp
class Solution {
public:
    ListNode* sortList(ListNode* head) {
	    // 特殊情况处理
        if (head == nullptr) {
            return head;
        }
        
        // 求链表长度
        int length = 0;
        ListNode* node = head;
        while (node != nullptr) {
            length++;
            node = node->next;
        }

		// 自行自底向上的排序
        ListNode* dummyHead = new ListNode(0, head);
        for (int subLength = 1; subLength < length; subLength <<= 1) {
            ListNode* prev = dummyHead, *curr = dummyHead->next;
            while (curr != nullptr) {
                ListNode* head1 = curr;
                for (int i = 1; i < subLength && curr->next != nullptr; i++) {
                    curr = curr->next;
                }
                ListNode* head2 = curr->next;
                curr->next = nullptr;
                curr = head2;
                for (int i = 1; i < subLength && curr != nullptr && curr->next != nullptr; i++) {
                    curr = curr->next;
                }
                ListNode* next = nullptr;
                if (curr != nullptr) {
                    next = curr->next;
                    curr->next = nullptr;
                }
                ListNode* merged = merge(head1, head2);
                prev->next = merged;
                while (prev->next != nullptr) {
                    prev = prev->next;
                }
                curr = next;
            }
        }
        return dummyHead->next;
    }

    ListNode* merge(ListNode* head1, ListNode* head2) {
        ListNode* dummyHead = new ListNode(0);
        ListNode* temp = dummyHead, *temp1 = head1, *temp2 = head2;
        while (temp1 != nullptr && temp2 != nullptr) {
            if (temp1->val <= temp2->val) {
                temp->next = temp1;
                temp1 = temp1->next;
            } else {
                temp->next = temp2;
                temp2 = temp2->next;
            }
            temp = temp->next;
        }
        if (temp1 != nullptr) {
            temp->next = temp1;
        } else if (temp2 != nullptr) {
            temp->next = temp2;
        }
        return dummyHead->next;
    }
};
```
可以看出，无论是哪种形式的归并算法，都非常依赖“合并两个有序链表”的操作。
当然，上面的算法想要在面试时手写是有一定难度的。所以这里再贴一种“邪道”做法，也就是直接通过数组排序完成的邪道做法。只可惜它需要$O(n)$的空间，在某些情况下可能会超出空间限制：
```java
public ListNode sortList(ListNode head) {
        int[] a = new int[5*10000];
        ListNode p = head;
        int i=0;
        // 拷贝到数组
        while (p != null) {
            a[i++] = p.val;
            p = p.next;
        }
        // 排序
        Arrays.sort(a,0,i);
        p = head;
        i = 0;
        // 拷贝到链表
        while(p!=null){
            p.val = a[i++];
            p = p.next;
        }
        return head;
    }
}
```
