给你一个字符串 s 。我们要把这个字符串划分为尽可能多的片段，同一字母最多出现在一个片段中。例如，字符串 "ababcc" 能够被分为 ["abab", "cc"]，但类似 ["aba", "bcc"] 或 ["ab", "ab", "cc"] 的划分是非法的。

注意，划分结果需要满足：将所有划分结果按顺序连接，得到的字符串仍然是 s 。

返回一个表示每个字符串片段的长度的列表。

提示：s 仅由小写英文字母组成

---
不得不说，我连题目都没看明白。这个“尽可能多的片段”和“同一字母最多出现在一个片段中”，我始终没有一点思路。
还是看看远处的[官方题解](https://leetcode.cn/problems/partition-labels/solutions/)吧。一看题解的第一句话，我就感觉自己太笨了，连这个都没有想到：
> 由于同一个字母只能出现在同一个片段，显然同一个字母的第一次出现的下标位置和最后一次出现的下标位置必须出现在同一个片段。因此需要遍历字符串，得到每个字母最后一次出现的下标位置。

这个看透事物本质的能力非常重要。
随后的解决方法就呼之欲出了。首先遍历一遍字符串，使用哈希表记录每个字符最后出现的位置，这里记这个哈希表为`dict`。然后再次遍历一遍字符串，假设此时遍历到的字符是`ch`，那么根据哈希表，这个字符最后出现的位置是`dict[ch]`，那么这个字符要被划分到的子串显然不能在`dict[ch]`之前截止。所以，我们不断地遍历字符串，并维护当前划分子串的右端点`r`，因为右端点要始终大于我们遍历中遇到的字符对应的最后一次出现的位置。当我们终于遍历到可行的右端点位置，即`i==r`时，我们立刻截止，并将当前的子串长度加入到结果数组。
写出的程序如下：
```cpp
class Solution {
public:
    vector<int> partitionLabels(string s) {
        // 记录每个字符最晚出现的位置
        unordered_map<char, int> pos;
        for (int i = 0; i < s.size(); i++) {
            pos[s[i]] = i;
        }

        // 开始划分
        vector<int> ans;
        int l = 0, r = 0; // 划分区间的左右端点
        for (int i = 0; i < s.size(); i++) {
            r = max(r, pos[s[i]]); // 该子串的右端点最左侧的可行值
            if (i == r) {
                ans.push_back(r - l + 1);
                l = r + 1;
            }
        }
        return ans;
    }
};
```
官方题解写出的程序与上面的程序几乎没有区别：
```cpp
class Solution {
public:
    vector<int> partitionLabels(string s) {
        int last[26];
        int length = s.size();
        for (int i = 0; i < length; i++) {
            last[s[i] - 'a'] = i;
        }
        vector<int> partition;
        int start = 0, end = 0;
        for (int i = 0; i < length; i++) {
            end = max(end, last[s[i] - 'a']);
            if (i == end) {
                partition.push_back(end - start + 1);
                start = end + 1;
            }
        }
        return partition;
    }
};
```
该算法的时间复杂度为$O(2n)=O(n)$，空间复杂度为$O(26)=O(1)$。
