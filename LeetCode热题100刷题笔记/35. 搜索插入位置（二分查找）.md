给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

请必须使用时间复杂度为 O(log n) 的算法。

提示：nums 为 无重复元素 的 升序 排列数组

---
本题是二分查找的最原始形态。二分查找的思想很简单，但是细节处的处理会比较麻烦，比如搜索区间究竟是用闭区间、开区间还是半开半闭区间表达，对应的循环条件都会有一些细节性的差别。下面直接给出一种最标准的基于**开区间**表达的程序：
```cpp
class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int l = -1, r = nums.size();
        while (l + 1 < r) {
            int mid = l + (r - l) / 2; // 防止数值溢出的写法
            if (nums[mid] == target)
                return mid;
            else if (nums[mid] > target)
                r = mid;
            else
                l = mid;
        }
        return l + 1;
    }
};
```
很容易分析出，该算法的时间复杂度为$O(\log n)$，空间复杂度为$O(1)$。
[官方题解](https://leetcode.cn/problems/search-insert-position/solutions/333632/sou-suo-cha-ru-wei-zhi-by-leetcode-solution/)则使用了闭区间表达，时间和空间的复杂度都是一样的。
```cpp
class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int n = nums.size();
        int left = 0, right = n - 1, ans = n;
        while (left <= right) {
            int mid = ((right - left) >> 1) + left;
            if (target <= nums[mid]) {
                ans = mid;
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        }
        return ans;
    }
};
```
