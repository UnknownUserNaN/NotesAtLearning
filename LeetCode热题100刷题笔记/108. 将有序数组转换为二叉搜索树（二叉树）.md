给你一个整数数组 nums ，其中元素已经按 升序 排列，请你将其转换为一棵 平衡 二叉搜索树。

平衡二叉树 是指该树所有节点的左右子树的高度相差不超过 1。

---
根据DeepSeek的解释，二叉搜索树具有以下核心特性：
> 1. 节点排列规则：每个节点的值均大于其左子树中所有节点的值，且小于其右子树中所有节点的值。
> 2. 递归结构：每个子树（左/右）本身也是一个二叉搜索树。
> 3. 中序遍历有序：对BST进行中序遍历（左 → 根 → 右），结果是一个升序排列的序列。

最后一条性质是关键。按这条性质，如果我们能拿到一个升序的序列，那么我们完全可以按照中序遍历的性质“反向构建”出二叉树。
本题要求构建的是**平衡**的二叉树，因此其左右子树的节点数一定是**近乎**相等的。这又是一个非常重要的性质。
对于此题，我们首先可以递归地想。给定一个升序数组，那么按照中序遍历的原则和平衡二叉树的性质，其最中间的元素一定是整个二叉树的“根”。以此元素为中心，其左侧的元素全部都是其左子树的结点，其右侧的元素全部都是其右子树的结点。然后我们便可以递归地考虑其左子树和右子树该如何构建，每次都是只取出其根元素，然后将剩余部分交给递归来构建。直到获得的子序列只有一个元素为止。
基于此思路，可以写出如下的代码：
```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left),
 * right(right) {}
 * };
 */
class Solution {
public:
    TreeNode* recursiveBuild(vector<int>& nums, int l, int r) {
        // 若传入的列表只有一个元素，则直接建立叶子结点
        if (l == r)
            return new TreeNode(nums[l]);

        // 获取最中间的元素，作为根节点
        int mid = (r + l) / 2;
        int mid_val = nums[mid];
        TreeNode* root = new TreeNode(mid_val);

        // 递归式地确定这个结点的左右孩子
        if (mid - 1 < l)
            root->left = nullptr;
        else
            root->left = recursiveBuild(nums, l, mid - 1);
        if (mid + 1 > r)
            root->right = nullptr;
        else
            root->right = recursiveBuild(nums, mid + 1, r);
        return root;
    }

    TreeNode* sortedArrayToBST(vector<int>& nums) {
        return recursiveBuild(nums, 0, nums.size() - 1);
    }
};
```
官方题解的写法要更简洁一些：
```cpp
class Solution {
public:
    TreeNode* sortedArrayToBST(vector<int>& nums) {
        return helper(nums, 0, nums.size() - 1);
    }

    TreeNode* helper(vector<int>& nums, int left, int right) {
        if (left > right) {
            return nullptr;
        }

        // 总是选择中间位置左边的数字作为根节点
        int mid = (left + right) / 2;

        TreeNode* root = new TreeNode(nums[mid]);
        root->left = helper(nums, left, mid - 1);
        root->right = helper(nums, mid + 1, right);
        return root;
    }
};
```
此种做法的时间复杂度为$O(n)$，空间复杂度与构成的树的深度有关，复杂度为$O(\log n)$。
上面的是写起来非常快速的递归方法，实际上面方法也可以改写成迭代方法（循环方法），只不过这种写法实在太庞杂了，要手动使用栈或者队列来模拟每次上面的`helper`函数或者`recursiveBuild`函数的参数输入，栈或者队列的内容将不得不设计成一个结构体。
