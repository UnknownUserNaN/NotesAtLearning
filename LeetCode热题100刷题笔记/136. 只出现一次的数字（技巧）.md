给你一个 非空 整数数组 nums ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

你必须设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常量额外空间。

提示：除了某个元素只出现一次以外，其余每个元素均出现两次。

---
初次碰到这个问题确实不知道该怎么实现满足题意的做法。但是好在题目给了提示，提醒考虑异或这个算符的性质。
对于本题而言，异或$\oplus$确实有一个非常重要的性质，即两个相同的数字做异或运算后会互相“抵消”，即$a\oplus a=0$。同时，异或又具有交换律，即$(a\oplus b)\oplus c = a\oplus (b\oplus c)$，这使得我们可以求取整个数组的异或和，由于其余的数字都出现两次，因此在交换律的保证下，这些重复出现的数字都会被“抵消”，唯独留下单独出现了一次的数字。
程序也非常简短：
```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int ret = 0;
        for (int num : nums)
            ret = ret ^ num;
        return ret;
    }
};
```
这个算法的时间复杂度是$O(n)$，空间复杂度是$O(1)$。不过很可惜，这种巧妙的算法，只能用于目标元素出现次数与其余元素出现次数恰好一个是奇数、一个是偶数的情况。
