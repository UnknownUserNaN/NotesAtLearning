给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。

**请注意** ，必须在不复制数组的情况下原地对数组进行操作。

***

既然题目提示了双指针，那本题肯定是要尽量在一次遍历中完成问题。一般情况下，**双指针中的其中一个往往会是数组遍历的指针，而另一个则会是与问题有关的某个指标**。

思路是这样的：遍历数组中的每个元素。如果当前元素是0，则计0的计数器（记为n）加一；如果当前元素不是0，则与此前的第n个元素交换位置。

这样，非0的元素会不断被往前“搬运”，而0的元素则会被逐渐搬运到后面。

我写的代码是这样的，性能排名居然达到了最优：

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int count_zero = 0;
        for(int i=0; i<nums.size(); i++){
            if(nums[i] == 0){
                count_zero++;
            }
            else{
                if(count_zero != 0){
                    int temp = nums[i];
                    nums[i] = nums[i-count_zero];
                    nums[i-count_zero] = temp;
                }
            }
        }
    }
};
```

官方题解的思路稍有不同，采用了“左指针”和“右指针”的说法，但实际方法与我的大同小异。

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int n = nums.size(), left = 0, right = 0;
        while (right < n) {
            if (nums[right]) {
                swap(nums[left], nums[right]);
                left++;
            }
            right++;
        }
    }
};
```

一个值得注意的C++语法点，就是swap函数可以直接调用，这样就省去写临时变量的步骤了。
