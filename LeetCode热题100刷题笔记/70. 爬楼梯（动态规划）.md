假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

---
这道题让我想到了一道经典的组合数学题：
> 有若干块1×2大小的骨牌，有多少种方式能使其铺满n×2大小的网格？

这道题的思路就与本题很类似。首先，考虑n×2的网格，铺到最后，骨牌一定要么占据1的长度，要么占据2的程度。因此，我们可以对n×2的网格进行“拆分”，一种方法将其拆分为(n-1)×2和1×2的两部分，另一种方法将其拆分为(n-2)×2和2×2的两部分。设题目所求情况的方法数是`dp[n]`，那么，前一种划分方案的方法数相当于`dp[n-1]*1`，即(n-1)×2的部分有`dp[n-1]`种密铺方法，剩下的1×2的部分只有一种密铺方法（骨牌竖着放）；后一种划分方案的方法数相当于`dp[n-2]*1`，即(n-2)×2的部分有`dp[n-2]`种密铺方法，剩下的2×2的部分只有一种密铺方法（只能是两张骨牌横着放，如果是竖着放，则相当于回归到了前一种划分方案的形式）。这样，总结下来就能得到递推式：
$$dp[n]=dp[n-1]+dp[n-2]$$
这是非常经典的斐波那契数列。不难发现`dp[1]=1`，`dp[2]=2`，于是剩下的部分可以直接通过该递推式求出。
本题的思路也是一样。假设到达第`n`阶的方法数为`dp[n]`，那么到达第`n`阶，我们可以通过跳1步和跳2步得到。如果是爬1步，那之前到达第`n-1`阶的步数就相当于`dp[n-1]`；如果是爬2步，那之前到达第`n-2`阶的步数就相当于`dp[n-2]`。综合之下，递推式（状态转移方程）与上面的形式完全相同。
不难注意到此时的边界条件`dp[1]=1`，`dp[2]=2`。基于此可以很快写出程序：
```cpp
class Solution {
public:
    int climbStairs(int n) {
        if (n == 1)
            return 1;
        if (n == 2)
            return 2;

        vector<int> dp(n);
        dp[0] = 1;
        dp[1] = 2;
        for (int i = 2; i < n; i++)
            dp[i] = dp[i - 1] + dp[i - 2];
        return dp[n - 1];
    }
};
```
容易分析出，该程序的时间复杂度为$O(n)$，空间复杂度为$O(n)$。
当然，由于本题是大名鼎鼎的斐波那契数列，所以本题完全可以采用斐波那契数列的通项公式来完成，这样就能达到近乎$O(1)$的时间复杂度和空间复杂度。下面是[官方题解](https://leetcode.cn/problems/climbing-stairs/solutions/286022/pa-lou-ti-by-leetcode-solution/)的写法：
```cpp
class Solution {
public:
    int climbStairs(int n) {
        double sqrt5 = sqrt(5);
        double fibn = pow((1 + sqrt5) / 2, n + 1) - pow((1 - sqrt5) / 2, n + 1);
        return (int)round(fibn / sqrt5);
    }
};
```
此外，官方题解还给出了一种矩阵快速幂的做法。关键思路是，他们基于上面的递推关系式，写出`dp[n]`和`dp[n+1]`与`dp[n-1]`和`dp[n]`之间的“矩阵表达的关系”：
$$\left[\begin{array}
&dp[n+1]\\
dp[n]
\end{array}\right]=
\left[\begin{array}
&dp[n]+dp[n-1]\\
dp[n-1]
\end{array}\right]=
\left[\begin{array}
&1 & 1\\
1 & 0
\end{array}\right]
\left[\begin{array}
&dp[n]\\
dp[n-1]
\end{array}\right]
$$
不断重复这个关系，直到达到`dp[0]`和`dp[1]`，就可以得到：
$$\left[\begin{array}
&dp[n+1]\\
dp[n]
\end{array}\right]=
\left[\begin{array}
&1 & 1\\
1 & 0
\end{array}\right]^n
\left[\begin{array}
&dp[1]\\
dp[0]
\end{array}\right]
$$
若记$M=\left[\begin{array}&1 & 1\\1 & 0\end{array}\right]$，这样如果我们能使用快速幂算法提前计算出上式中的$M^n$，便可以快速得到上面递推式的结果。
下面是官方题解的程序：
```cpp
class Solution {
public:
    vector<vector<long long>> multiply(vector<vector<long long>> &a, vector<vector<long long>> &b) {
        vector<vector<long long>> c(2, vector<long long>(2));
        for (int i = 0; i < 2; i++) {
            for (int j = 0; j < 2; j++) {
                c[i][j] = a[i][0] * b[0][j] + a[i][1] * b[1][j];
            }
        }
        return c;
    }

    vector<vector<long long>> matrixPow(vector<vector<long long>> a, int n) {
        vector<vector<long long>> ret = {{1, 0}, {0, 1}};
        while (n > 0) {
            if ((n & 1) == 1) {
                ret = multiply(ret, a);
            }
            n >>= 1;
            a = multiply(a, a); // 基于二分法的快速幂
        }
        return ret;
    }

    int climbStairs(int n) {
        vector<vector<long long>> ret = {{1, 1}, {1, 0}};
        vector<vector<long long>> res = matrixPow(ret, n);
        return res[0][0];
    }
};
```
快速幂算法的核心思想是递归和二分法。若$n$为偶数，$a^n=a^{n/2}a^{n/2}$；若$n$为奇数，$a^n=a^{n-1/2}a^{n-1/2}a$。剩余部分的幂都可以采用递归方法求取，直到达到自身的0次幂和1次幂。
该算法的时间复杂度由此达到了$O(\log n)$，空间复杂度也由此达到$O(1)$。
此种方法对遇到齐次递推式的方法都非常好用。对于非齐次递推式的情况，则需要设法运用线性代数的方法设新的变量，对原递推式进行齐次化。
