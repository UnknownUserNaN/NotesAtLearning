给你一个满足下述两条属性的 m x n 整数矩阵：
- 每行中的整数从左到右按非严格递增顺序排列。
- 每行的第一个整数大于前一行的最后一个整数。
给你一个整数 target ，如果 target 在矩阵中，返回 true ；否则，返回 false 。

---
一个最简单的思路是使用两次二分查找，一次二分查找用来找目标在哪一行，另一次二分查找用来找目标在这一行的哪一个位置。当然也可以简单粗暴地把这个二维矩阵拼接成一个一维的数组然后应用一维的二分查找，但是这样需要额外的空间，不经济。
熟悉二分查找的模板的话很容易写出对应程序，但是实际应用的时候很难一次性直接跑通。为了减少不必要的烦恼，查询时最好还是采用开区间表示搜索区间（虽然下面的程序实际上是用了开区间和闭区间的两种表达）。
```cpp
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        // 先搜索每行的开头，看目标可能在哪一行
        int l = -1, r = matrix.size(); // 开区间表示
        while (l + 1 < r) {
            int mid = l + (r - l) / 2;
            if (matrix[mid][0] == target)
                return true;
            else if (matrix[mid][0] > target)
                r = mid;
            else
                l = mid;
        }

        // 搜索完毕，目标最可能在l行
        int line = max(0, l); // 防止出界，因为有时左指针可能根本没有动过
        l = 0;
        r = matrix[0].size() - 1; // 闭区间表示
        while (l < r) {
            int mid = l + (r - l) / 2;
            if (matrix[line][mid] == target)
                return true;
            else if (matrix[line][mid] > target)
                r = mid - 1;
            else
                l = mid + 1;
        }
        return matrix[line][l] == target;
    }
};
```
[官方题解](https://leetcode.cn/problems/search-a-2d-matrix/solutions/688117/sou-suo-er-wei-ju-zhen-by-leetcode-solut-vxui/)则采用了C++的匿名函数和库函数（如upper_bound和binary_search）来完成本题，写起来更为简洁：
```cpp
class Solution {
public:
    bool searchMatrix(vector<vector<int>> matrix, int target) {
        auto row = upper_bound(matrix.begin(), matrix.end(), target, [](const int b, const vector<int> &a) {
            return b < a[0];
        });
        if (row == matrix.begin()) {
            return false;
        }
        --row;
        return binary_search(row->begin(), row->end(), target);
    }
};
```
该方法的时间复杂度为$O(\log mn)$，空间复杂度为$O(1)$。
官方题解对前面的一次二分查找方法也进行了优化：实际上我们并不需要真正地拼接矩阵，只需要将这个一维数组的下标想办法映射成二维数组中的行和列就可以了。
```cpp
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        int m = matrix.size(), n = matrix[0].size();
        int low = 0, high = m * n - 1;
        while (low <= high) {
            int mid = (high - low) / 2 + low;
            int x = matrix[mid / n][mid % n]; // 将一维数组中的下标转化为二维矩阵的行和列
            if (x < target) {
                low = mid + 1;
            } else if (x > target) {
                high = mid - 1;
            } else {
                return true;
            }
        }
        return false;
    }
};
```
其复杂度与之前是相同的。
