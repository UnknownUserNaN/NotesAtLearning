给你一个整数数组 nums ，数组中的元素 互不相同 。返回该数组所有可能的子集（幂集）。

解集 不能 包含重复的子集。你可以按 任意顺序 返回解集。

提示：nums 中的所有元素 互不相同

---
按照回溯法的一般模板，这个问题很容易得到思路。
考虑回溯法所对应的树形结构，整个问题构成一个二叉树，每个深度为$i$的结点表示该位置的元素是否可以加入结果结合，向左为不加入该元素，向右为加入，如此构成一个深度为$N$的二叉树（$N$为数组nums的长度），回溯法即完成该树的深度遍历。每次遍历到叶子结点后，即将对应的子集加入结果集合。
初次写出的程序运行效率稍低，可能是差在了来回增删数组的操作上了。
```cpp
class Solution {
public:
    void backtrack(vector<vector<int>>& ret, vector<int>& nums,
                   vector<int>& selected, int n_chosen) {
        // 已经达到可以反向回归的结点
        if (n_chosen == nums.size()) {
            vector<int> result;
            for (int i : selected) {
                result.push_back(nums[i]);
            }
            ret.push_back(result);
            return;
        }

        // 选择或不选择添入当前要选择的元素
        backtrack(ret, nums, selected, n_chosen + 1); // 先不选中当前元素
        selected.push_back(n_chosen);
        backtrack(ret, nums, selected, n_chosen + 1); // 再选中当前元素
        selected.pop_back();
    }

    vector<vector<int>> subsets(vector<int>& nums) {
        vector<vector<int>> ret;
        vector<int> selected;
        backtrack(ret, nums, selected, 0);
        return ret;
    }
};
```
[官方题解](https://leetcode.cn/problems/subsets/solutions/420294/zi-ji-by-leetcode-solution/)的做法则更为简洁，直接通过相应元素的增删来组成答案。
```cpp
class Solution {
public:
    vector<int> t;
    vector<vector<int>> ans;

    void dfs(int cur, vector<int>& nums) {
        if (cur == nums.size()) {
            ans.push_back(t);
            return;
        }
        t.push_back(nums[cur]);
        dfs(cur + 1, nums);
        t.pop_back();
        dfs(cur + 1, nums);
    }

    vector<vector<int>> subsets(vector<int>& nums) {
        dfs(0, nums);
        return ans;
    }
};
```
该算法的复杂度分析可以通过对二叉树的结点计数进行。容易得到，它的时间复杂度为$O(n\cdot2^n)$，空间复杂度为$O(n)$。
