给定一个二叉树 root ，返回其最大深度。

二叉树的 最大深度 是指从根节点到最远叶子节点的最长路径上的节点数。

---
这里就是经典的两种思路：深度优先（DFS）和广度优先（BFS）。
深度优先思路最简单，也最容易想到。遇到一个结点，优先向着更深处“走”，每往深处走一次，就将深度加一，最后设法维护这个遍历过程中深度的最大值即可。这实际上只需要在二叉树遍历的程序上稍作修改即可，比如下面是在二叉树的前序遍历的基础上修改得到的：
```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left),
 * right(right) {}
 * };
 */
class Solution {
public:
    int maxDepth(TreeNode* root) {
        if (!root)
            return 0;

        stack<pair<TreeNode*, int>> st; // 栈存储 <节点, 当前深度>
        st.push({root, 1});
        int max_depth = 1;

        while (!st.empty()) {
            auto [node, depth] = st.top();
            st.pop();

            max_depth = max(max_depth, depth); // 更新最大深度

            // 前序位置：先右后左入栈，保证左子树先处理
            if (node->right)
                st.push({node->right, depth + 1});
            if (node->left)
                st.push({node->left, depth + 1});
        }
        return max_depth;
    }
};
```
当然也可以用递归的方法来完成这个问题，实际上递归的思路更加简单。如果我们已经知道了当前结点的左子树和右子树的深度，那么当前结点的深度实际上就是两个子树深度较大者的值加一。写成程序只有甚至只有寥寥几行：
```cpp
class Solution {
public:
    int maxDepth(TreeNode* root) {
        if (root == nullptr)
            return 0;
        else
            return max(maxDepth(root->left), maxDepth(root->right)) + 1;
    }
};
```
这种做法的时间复杂度是$O(n)$，因为这需要遍历整个二叉树，空间复杂度是$O(d)$，其中$d$为二叉树的深度，最坏情况下，也就是二叉树恰好只有一条时，此时$d=n$，空间复杂度达到$O(n)$。
然后是广度优先，这个算法不得不采用额外的容器来存储状态。具体思路是，首先将根节点放入容器，然后将容器内的所有元素向“深处”“扩展”一次，每“扩展”一次，就将深度加一。不断重复这个“扩展”的步骤，直到容器内的所有元素均无法再“扩展”为止。
为了方便操作，上面的容器最好使用队列实现。
```cpp
class Solution {
public:
    int maxDepth(TreeNode* root) {
        if (root == nullptr)
            return 0; // 特殊情况处理
            
        queue<TreeNode*> nodes;
        nodes.push(root);
        int maxdepth = 1;
        while (!nodes.empty()) {
            int n = nodes.size();
            for (int i = 0; i < n; i++) {
                if (nodes.front()->left != nullptr)
                    nodes.push(nodes.front()->left);
                if (nodes.front()->right != nullptr)
                    nodes.push(nodes.front()->right);
                nodes.pop();
            }
            maxdepth++;
        }
        return maxdepth - 1;
    }
};
```
这种做法的时间复杂度是$O(n)$，因为这需要遍历整个二叉树，空间复杂度是$O(w)$，其中$w$为二叉树“一层”中最大的“宽度”。在最坏情况下，也就是二叉树的末端有一层非常宽时，可以近似认为$w\approx n$，此时该算法要使用$O(n)$的空间。
