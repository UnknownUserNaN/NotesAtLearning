给你一个二叉树的根节点 root ，判断其是否是一个有效的二叉搜索树。
有效 二叉搜索树定义如下：
- 节点的左子树只包含 小于 当前节点的数。
- 节点的右子树只包含 大于 当前节点的数。
- 所有左子树和右子树自身必须也是二叉搜索树。

---
二叉搜索树的一条重要性质是，其中序遍历的结果必定是一个升序数组。因此，我们完全可以在原来中序遍历程序的基础上稍作修改，得到验证二叉搜索树的版本。
下面是在经典的Morris算法上修改后得到的版本。但是很遗憾，这个修改后的程序会莫名其妙在在线OJ上造成栈溢出。造成栈溢出的行已经在下面进行了标注。
```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left),
 * right(right) {}
 * };
 */
class Solution {
public:
    bool Morris(TreeNode* root) {
        int prev_val = INT_MIN;
        TreeNode *p = root, *pre = nullptr;

        while (p != nullptr) {
            if (p->left == nullptr) {
                if (p->val <= prev_val)
                    return false;
                prev_val = p->val;
                p = p->right;
            } else {
                pre = p->left;
                while (pre->right != nullptr && pre->right != p)
                    pre = pre->right;

                if (pre->right == nullptr) {
                    pre->right = p;
                    p = p->left;
                } else {
                    if (p->val <= prev_val)
                        return false;
                    prev_val = p->val; // 如果不注释，会报错的一行
                    pre->right = nullptr;
                    p = p->right;
                }
            }
        }
        return true;
    }

    bool isValidBST(TreeNode* root) { return Morris(root); }
};
```
改用经典的递归方法遍历后，就不会造成栈溢出的问题：
```cpp
class Solution {
    // int prev_val = INT_MIN; // 采用此法易导致无法处理结点数值恰为INT_MIN的情况
    long prev_val = LONG_MIN; 
    bool flag = true;

public:
    void inorderTrace(TreeNode* root) {
        if (root->left != nullptr)
            inorderTrace(root->left);

        if (root->val <= prev_val) {
            flag = false;
            return;
        } else
            prev_val = root->val;

        if (root->right != nullptr)
            inorderTrace(root->right);
    }

    bool isValidBST(TreeNode* root) {
        if (root->left == nullptr && root->right == nullptr)
            return true;

        inorderTrace(root);
        return flag;
    }
};
```
这种做法的复杂度与递归遍历二叉树的复杂度一样，时间复杂度为$O(n)$，空间复杂度最差为$O(n)$。
当然，本题也可以按照题目暗示，通过递归方法完成。对于一个结点，我们首先判断它的左子树和右子树是否为二叉搜索树。如果是，则寻找该左子树中的最大值`l`和右子树中的最小值`r`，如果当前结点的值`v`满足`l<v<r`，则说明以当前结点为根的二叉树也符合要求。两个子树中的最值很好求取，因为两个子树都是二叉搜索树，所以其最大值一定在最右侧的位置，最小值一定在最左侧的位置。程序如下：
```cpp
class Solution {
public:
    bool recursiveCheck(TreeNode* root) {
        if (root->left == nullptr && root->right == nullptr)
            return true;
        if (root->left != nullptr && !recursiveCheck(root->left))
            return false;
        if (root->right != nullptr && !recursiveCheck(root->right))
            return false;

        TreeNode *pl_max = root->left, *pr_min = root->right;
        if (pl_max != nullptr) {
            while (pl_max->right != nullptr)
                pl_max = pl_max->right;
            if (root->val <= pl_max->val)
                return false;
        }

        if (pr_min != nullptr) {
            while (pr_min->left != nullptr)
                pr_min = pr_min->left;
            if (root->val >= pr_min->val)
                return false;
        }

        return true;
    }

    bool isValidBST(TreeNode* root) { return recursiveCheck(root); }
};
```
这种算法的时间复杂度也为$O(n)$，最坏情况下的空间复杂度也为$O(n)$。
