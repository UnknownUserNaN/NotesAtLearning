给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s ，判断字符串是否有效。

有效字符串需满足：

- 左括号必须用相同类型的右括号闭合。
- 左括号必须以正确的顺序闭合。
- 每个右括号都有一个对应的相同类型的左括号。

提示：s 仅由括号 '()\[\]{}' 组成

---
这是一道非常经典的用栈解决的问题。基本的思路是，遍历字符串，当遍历到左括号时，将左括号入栈；当遍历到右括号时，检测栈顶元素是否与该括号匹配，若不匹配则直接返回否，若匹配则将栈顶元素出栈——当然，如果此时不存在栈顶元素，则也可以直接返回否。如果遍历完字符串后栈非空，则返回否，否则说明字符串可以完成括号匹配，返回是。
很容易写出如下的程序：
```cpp
class Solution {
public:
    bool isValid(string s) {
        stack<char> stk;
        for (char c : s) {
            switch (c) {
            // 为左括号，直接将该元素入栈
            case '(':
            case '[':
            case '{':
                stk.push(c);
                break;
            // 为右括号，检测与栈顶元素是否匹配
            case ')':
                if (stk.empty() || stk.top() != '(')
                    return false;
                stk.pop();
                break;
            case ']':
                if (stk.empty() || stk.top() != '[')
                    return false;
                stk.pop();
                break;
            case '}':
                if (stk.empty() || stk.top() != '{')
                    return false;
                stk.pop();
                break;
            }
        }
        return stk.size() == 0;
    }
};
```
[官方题解](https://leetcode.cn/problems/valid-parentheses/solutions/373578/you-xiao-de-gua-hao-by-leetcode-solution/)的做法与上面的思路大同小异，但特别之处在于使用了哈希表来存储对应的括号，从而使得程序更为精简。
```cpp
class Solution {
public:
    bool isValid(string s) {
        int n = s.size();
        if (n % 2 == 1) {
            return false;
        }

        unordered_map<char, char> pairs = {
            {')', '('},
            {']', '['},
            {'}', '{'}
        };
        stack<char> stk;
        for (char ch: s) {
            if (pairs.count(ch)) {
                if (stk.empty() || stk.top() != pairs[ch]) {
                    return false;
                }
                stk.pop();
            }
            else {
                stk.push(ch);
            }
        }
        return stk.empty();
    }
};
```
上述做法的时间复杂度为$O(n)$，空间复杂度也为$O(n)$。
