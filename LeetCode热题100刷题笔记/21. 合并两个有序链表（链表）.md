将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

---
思路依然很简单。基本是遍历第一个链表，如果第一个链表中有适合插入的位置，则将第二个链表中的元素插入到相应位置，并改变相应元素的连接关系。其中需要注意的是，在开始遍历链表之前还要检查能否将第二个链表的内容插入到第一个链表之前（并相应移动链表的头指针）；在遍历完第一个链表之后，如果第二个链表还有元素未插入，那么就需要将第二个链表的内容链接到第一个链表的末尾。
总之这个问题看似简单，但实际上有许多细节需要处理。最后写出了下面的代码，有些丑陋：
```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* mergeTwoLists(ListNode* list1, ListNode* list2) {
        // 例外情况处理
        if (list1 == nullptr && list2 != nullptr)
            return list2;
        else if (list1 != nullptr && list2 == nullptr)
            return list1;
        else if (list1 == nullptr && list2 == nullptr)
            return nullptr;

        // 遍历第一个链表，观察是否有适合插入第二个链表内容的位置
        ListNode *p1 = list1, *p2 = list2;
        ListNode* temp;
        while (p1 != nullptr && p2 != nullptr && p2->val <= p1->val) {
            // 将p2连接至list1开头
            temp = p2->next;
            list1 = p2;
            p2->next = p1;
            p1 = p2;
            p2 = temp;
        }
        while (p1->next != nullptr) {
            if (p2 != nullptr && p2->val >= p1->val &&
                p2->val <= p1->next->val) {
                // list1中有适当位置可以插入list2中的元素
                temp = p2->next;
                p2->next = p1->next;
                p1->next = p2;
                p1 = p2;
                p2 = temp;
            } else
                p1 = p1->next;
        }
        if (p2 != nullptr) // list2中仍有元素等待插入
            p1->next = p2;

        return list1;
    }
};
```
[官方题解](https://leetcode.cn/problems/merge-two-sorted-lists/solutions/226408/he-bing-liang-ge-you-xu-lian-biao-by-leetcode-solu/)的思路就严谨很多：
> 首先，我们设定一个哨兵节点 prehead ，这可以在最后让我们比较容易地返回合并后的链表。我们维护一个 prev 指针，我们需要做的是调整它的 next 指针。然后，我们重复以下过程，直到 l1 或者 l2 指向了 null ：如果 l1 当前节点的值小于等于 l2 ，我们就把 l1 当前的节点接在 prev 节点的后面同时将 l1 指针往后移一位。否则，我们对 l2 做同样的操作。不管我们将哪一个元素接在了后面，我们都需要把 prev 向后移一位。
   在循环终止的时候， l1 和 l2 至多有一个是非空的。由于输入的两个链表都是有序的，所以不管哪个链表是非空的，它包含的所有元素都比前面已经合并链表中的所有元素都要大。这意味着我们只需要简单地将非空链表接在合并链表的后面，并返回合并链表即可。

这里需要注意的是，官方题解采用的prev指针会根据情况，在list1和list2之间来回“跳”，这能使得两个链表能通过最少的操作连接在一起。这里非常推荐观看官方题解的可视化解法。
基于这个思路，写出的代码也非常简洁：
```cpp
class Solution {
public:
    ListNode* mergeTwoLists(ListNode* l1, ListNode* l2) {
        ListNode* preHead = new ListNode(-1);

        ListNode* prev = preHead;
        while (l1 != nullptr && l2 != nullptr) {
            if (l1->val < l2->val) {
                prev->next = l1;
                l1 = l1->next;
            } else {
                prev->next = l2;
                l2 = l2->next;
            }
            prev = prev->next;
        }

        // 合并后 l1 和 l2 最多只有一个还未被合并完，我们直接将链表末尾指向未合并完的链表即可
        prev->next = l1 == nullptr ? l2 : l1;

        return preHead->next;
    }
};
```
