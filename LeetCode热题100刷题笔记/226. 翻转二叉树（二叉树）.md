给你一棵二叉树的根节点 root ，翻转这棵二叉树，并返回其根节点。

---
翻转二叉树，要求交换所有节点的左右孩子。
二叉树的许多基本操作都可以通过递归方法实现。比如本题就可以通过类似深度优先的递归方法实现：
```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left),
 * right(right) {}
 * };
 */
class Solution {
public:
    TreeNode* invertTree(TreeNode* root) {
        if (root != nullptr) {
            if (root->left != nullptr)
                invertTree(root->left);
            if (root->right != nullptr)
                invertTree(root->right);
            swap(root->left, root->right);
        }
        return root;
    }
};
```
广度优先的思路也可以实现。我们不断扩展容器中的所有元素，并均交换其左右孩子。写出的程序如下：
```cpp
class Solution {
public:
    TreeNode* invertTree(TreeNode* root) {
        if (root == nullptr)
            return 0; // 特殊情况处理

        queue<TreeNode*> nodes;
        nodes.push(root);
        while (!nodes.empty()) {
            int n = nodes.size();
            for (int i = 0; i < n; i++) {
                if (nodes.front()->left != nullptr)
                    nodes.push(nodes.front()->left);
                if (nodes.front()->right != nullptr)
                    nodes.push(nodes.front()->right);
                swap(nodes.front()->left, nodes.front()->right);
                nodes.pop();
            }
        }
        return root;
    }
};
```
这两种算法的复杂度分析均与之前的相似，详细可参考104题。这两种算法的时间复杂度均为$O(n)$，最坏情况下的空间复杂度均为$O(n)$。
