给定一个包含红色、白色和蓝色、共 n 个元素的数组 nums ，原地 对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

我们使用整数 0、 1 和 2 分别表示红色、白色和蓝色。

必须在不使用库内置的 sort 函数的情况下解决这个问题。

提示：nums[i] 为 0、1 或 2

进阶：你能想出一个仅使用常数空间的一趟扫描算法吗？

---
本题的提示“剧透”了本题的方法。既然本题中只有0, 1, 2三种元素，我们便可以使用计数排序的方法：即遍历一遍数组，数出数组中0, 1, 2元素的个数；然后再遍历一遍数组，根据前面计数的结果分别将数组元素覆写。
很容易写出如下的程序：
```cpp
class Solution {
public:
    void sortColors(vector<int>& nums) {
        // 先计数
        int counter[3] = {0};
        for (int num : nums) {
            counter[num]++;
        }

        // 再覆写
        int cur = 0;
        for (int i = 0; i < 3; i++) {
            while (counter[i] > 0) {
                nums[cur] = i;
                cur++;
                counter[i]--;
            }
        }
    }
};
```
上面的程序对数组进行了两次扫描。但是题目的进阶部分提示我们，应当存在遍历一次就可以完成的算法。
这里很容易结合双指针的方法想到一种思路。建立一个与输入数组同形的数组，数组中预先填入的只有1。遍历输入的数组，如果遇到了0，则在该数组开头插入0，然后插入的指针向右移动一位；如果遇到了2，则在该数组末尾插入2，然后指针向左移动一位；如果遇到1则不作处理。
当然这个思路其实可以再做简化，我们可以利用交换（swap）这个操作来完成原地算法。即当我们遍历到0时，我们设法将该元素交换到数组开头；当我们遍历到2时，则设法将该元素交换到数组末尾；遍历到1则不做处理。
很容易写出下面的程序：
```cpp
class Solution {
public:
    void sortColors(vector<int>& nums) {
        int left = 0, right = nums.size() - 1;
        int i = 0;
        while (i <= right) {
            if (nums[i] == 0) {
                swap(nums[i], nums[left]);
                i++;
                left++;
            } else if (nums[i] == 2) {
                swap(nums[i], nums[right]);
                right--; // 我们不知道这个元素究竟为何数，所以需要不能增加遍历的下标
            } else
                i++;
        }
    }
};
```
这样，时间复杂度就降到了$O(n)$，空间复杂度就降到了$O(1)$。
[官方题解](https://leetcode.cn/problems/sort-colors/solutions/437968/yan-se-fen-lei-by-leetcode-solution/)还给出了另一种双指针的做法。思路是，从左向右遍历时，如果遇到了1，则设法将其往数组的“中间”搬运；遇到了0，则设法将其往数组的开头搬运。当然，这个“中间”的定义是，所有的1必须出现在所有的0后面，因此我们需要一边遍历，一边记录0和1的个数。如果搬运0时”误伤“了1，则将被”误伤“的1搬运到目前所有的1的最后。
```cpp
class Solution {
public:
    void sortColors(vector<int>& nums) {
        int n = nums.size();
        int p0 = 0, p1 = 0;
        for (int i = 0; i < n; ++i) {
            if (nums[i] == 1) {
                swap(nums[i], nums[p1]);
                ++p1;
            } else if (nums[i] == 0) {
                swap(nums[i], nums[p0]);
                if (p0 < p1) {
                    swap(nums[i], nums[p1]);
                }
                ++p0;
                ++p1;
            }
        }
    }
};
```
