给定一个单链表 L 的头节点 head ，单链表 L 表示为：

L0 → L1 → … → Ln - 1 → Ln
请将其重新排列后变为：

L0 → Ln → L1 → Ln - 1 → L2 → Ln - 2 → …
不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

---
如果能注意到，最终重排得到的链表实际上是链表的开头和末尾不断向中间合拢后得到的，就不难想到这个题该怎么做了。官方题解将此题的做法拆分成了链表的三个基础操作，能够有效地降低编程难度：
1. 查找链表中点，以此将链表拆分成两个部分。这里可以使用**快慢指针法**快速得到。当然这里拆分得到的两个链表不一定是严格等长的，但两者的长度最多也就相差1，完全在可以接受的范围内。
2. 从链表中点开始，对后半部分的**链表倒序**。这里就是经典的三指针法，循环外是指针`prev`和`curr`，循环内是指针`next`，用来记录`curr`原定指向的下一个元素。循环条件是`curr != nullptr`。
3. 将上面的两个链表进行合并。由于这里两个链表的长度仅相差一，因此链表的合并可以比之前的程序写得更加简单。只需要依次将两个链表的结点依次连接到末尾的结点上即可。
官方题解给出的程序也非常的简洁。`getMid`、`reverseList`和`merge2`两个函数分别用来实现上面的1~3步：
```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* getMid(ListNode* head){
        ListNode* slow = head, *fast = head;
        while(fast->next!=nullptr && fast->next->next!=nullptr){
            fast = fast->next->next;
            slow = slow->next;
        }
        return slow;
    }

    ListNode* reverseList(ListNode* head){
        ListNode* curr = head, *prev = nullptr;
        while(curr!=nullptr){
            ListNode* temp = curr->next;
            curr->next = prev;
            prev = curr;
            curr = temp;
        }
        return prev;
    }

    void merge2(ListNode* l1, ListNode* l2){
        ListNode *p1=l1, *p2=l2;
        while(p1!=nullptr && p2!=nullptr){
            p1 = l1->next;
            p2 = l2->next;

            l1->next = l2;
            l1 = p1;

            l2->next = l1;
            l2 = p2;
        }
    }

    void reorderList(ListNode* head) {
        if(head == nullptr) return;

        ListNode *mid = getMid(head);
        ListNode *l1 = head;
        ListNode *l2 = mid->next;
        mid->next = nullptr;

        l2 = reverseList(l2);
        return merge2(l1, l2);
    }
};
```
这样的程序大致需要遍历三遍链表，时间复杂度为$O(3n)=O(n)$；只使用了常数级的额外空间，所以是$O(1)$的空间复杂度。
