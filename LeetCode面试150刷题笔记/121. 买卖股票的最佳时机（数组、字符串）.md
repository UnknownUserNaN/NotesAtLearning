给定一个数组 `prices` ，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。

你只能选择 **某一天** 买入这只股票，并选择在 **未来的某一个不同的日子** 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0` 。

***

这题与之前在热题100做过的一道题思路类似。首先基于枚举的思路，对于每个被枚举的元素，不断计算其与此前遇到过的最小值的差。然后在遍历过程中，记录遇到的这个差的最大值。这是基本思路，区别在于，如果这个差是负数（结合情景，也就是赚不到钱），那么直接返回0，表示根本没有买入和卖出的机会。

基于上面这个思路，可以写出如下的程序：

```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int min_price = INT_MAX;
        int profit = 0;
        int max_profit = INT_MIN;
        for(int price: prices){
            profit = max(0, price-min_price); // 先计算该位置的差价
            max_profit = max(profit, max_profit); // 再维护差价的最大值
            min_price = min(price, min_price); // 维护此前遇到的最小值
        }
        return max_profit;
    }
};
```
