编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 `""`。

***

想法很简单。先拿数组中的第一个字符串与第二个做逐个比较，能匹配上的字符按顺序保留，字符不能匹配上直接结束匹配。按此顺序不断遍历数组，这样就可以进行数组的一次遍历便找到最长的公共前缀：

```cpp
class Solution {
public:
    string longestCommonPrefix(vector<string>& strs) {
        string pre = strs[0];

        for (int i = 1; i < strs.size(); i++) {
            string next = "";
            for (int j = 0; j < min(pre.size(), strs[i].size()); j++) {
                if (pre[j] != strs[i][j])
                    break;
                next += pre[j];
            }
            pre = next;
        }

        return pre;
    }
};
```

官方题解除了上面的“横向查找”以外，还给出了“纵向查找”的方法。也就是先查找每个字符串的第一个字符，如果第一个字符都相同，则将该字符整合到输出结果中，然后查找每个字符串的第二个字符……

```cpp
class Solution {
public:
    string longestCommonPrefix(vector<string>& strs) {
        if (!strs.size()) {
            return "";
        }
        int length = strs[0].size();
        int count = strs.size();
        for (int i = 0; i < length; ++i) {
            char c = strs[0][i];
            for (int j = 1; j < count; ++j) {
                if (i == strs[j].size() || strs[j][i] != c) {
                    return strs[0].substr(0, i);
                }
            }
        }
        return strs[0];
    }
};
```

除此之外，官方还给出了分治法和二分法的做法，但是这些做法的时间和空间复杂度都往往大于前两种方法。
