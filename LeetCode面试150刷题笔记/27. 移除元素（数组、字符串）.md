给你一个数组 `nums`\* *和一个值 `val`，你需要 **[原地](https://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95)** 移除所有数值等于 `val`* \*的元素。元素的顺序可能发生改变。然后返回 `nums` 中与 `val` 不同的元素的数量。

假设 `nums` 中不等于 `val` 的元素数量为 `k`，要通过此题，您需要执行以下操作：

*   更改 `nums` 数组，使 `nums` 的前 `k` 个元素包含不等于 `val` 的元素。`nums` 的其余元素和 `nums` 的大小并不重要。
*   返回 `k`。

整个思路依旧非常简单。遍历数组中的每个元素，如果当前元素恰好与要删除的元素相同，则将该元素后侧的元素复制过来覆盖掉此片区域。如果最终将数组中的所有元素都遍历完了，那么该程序直接结束。

基于此思路可以写出下面的代码：

```cpp
class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        int valid_pos = nums.size() - 1;
        int remove_times = 0;
        int i = 0;
        while (i <= valid_pos) {
            while (nums[i + remove_times] == val) {
                remove_times++;
            }
            nums[i] = nums[i + remove_times];
            valid_pos = nums.size() - remove_times - 1;
            i++;
        }
        return nums.size() - remove_times;
    }
};
```

这一行不对，因为语法检查时会认为这样有越界访问的危险。最后给出了更为简洁做法的代码：

```cpp
class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        int j = 0;  // 记录下一个有效元素的位置
        for (int i = 0; i < nums.size(); i++) {
            if (nums[i] != val) {
                nums[j] = nums[i];
                j++;
            }
        }
        return j;  // j 是去除 val 之后的新数组长度
    }
};
```

其实思路大同小异，但是更加简洁。只需要注意变量j是用来记录真正**有效元素**的位置即可，整个思路就是遍历数组，然后往有效的位置填入不是要删除的元素的数值即可。
