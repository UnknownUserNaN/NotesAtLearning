实现`RandomizedSet` 类：

*   `RandomizedSet()` 初始化 `RandomizedSet` 对象
*   `bool insert(int val)` 当元素 `val` 不存在时，向集合中插入该项，并返回 `true` ；否则，返回 `false` 。
*   `bool remove(int val)` 当元素 `val` 存在时，从集合中移除该项，并返回 `true` ；否则，返回 `false` 。
*   `int getRandom()` 随机返回现有集合中的一项（测试用例保证调用此方法时集合中至少存在一个元素）。每个元素应该有 **相同的概率** 被返回。

你必须实现类的所有函数，并满足每个函数的 **平均** 时间复杂度为 `O(1)` 。

 

**示例：**

    输入
    ["RandomizedSet", "insert", "remove", "insert", "getRandom", "remove", "insert", "getRandom"]
    [[], [1], [2], [2], [], [1], [2], []]
    输出
    [null, true, false, true, 2, true, false, 2]

    解释
    RandomizedSet randomizedSet = new RandomizedSet();
    randomizedSet.insert(1); // 向集合中插入 1 。返回 true 表示 1 被成功地插入。
    randomizedSet.remove(2); // 返回 false ，表示集合中不存在 2 。
    randomizedSet.insert(2); // 向集合中插入 2 。返回 true 。集合现在包含 [1,2] 。
    randomizedSet.getRandom(); // getRandom 应随机返回 1 或 2 。
    randomizedSet.remove(1); // 从集合中移除 1 ，返回 true 。集合现在包含 [2] 。
    randomizedSet.insert(2); // 2 已在集合中，所以返回 false 。
    randomizedSet.getRandom(); // 由于 2 是集合中唯一的数字，getRandom 总是返回 2 。

 

**提示：**

*   `-2`^31^` <= val <= 2`^31^` - 1`
*   最多调用 `insert`、`remove` 和 `getRandom` 函数 `2 * 10`^5^ 次
*   在调用 `getRandom` 方法时，数据结构中 **至少存在一个** 元素。

由于题目要求各个成员函数的时间复杂度都能达到`$O(1)$`的时间复杂度，能达到如此低的复杂度，我最先想到的就是哈希表这种形式。采用哈希表（更准确地说，应该是unordered\_map这种容器）这种形式，能非常容易地实现`$O(1)$`时间内的查找等操作。

基于这个容器，就可以写出下面的代码：

```cpp
class RandomizedSet {
private:
    unordered_map<int, int> m;

public:
    RandomizedSet() {}

    bool insert(int val) {
        if (this->m[val] < 1) { // 查询该元素是否存在
            this->m[val] = 1;
            return true;
        } else {
            this->m[val] = 1;
            return false;
        }
    }

    bool remove(int val) {
        if (this->m[val] >= 1) { // 查询该元素是否存在
            this->m[val] = 0;
            return true;
        } else {
            this->m[val] = 0;
            return false;
        }
    }

    int getRandom() {
        auto iter = this->m.begin();
        return (*iter).first;
    }
};
```

但是，上面的代码并没有通过所有的测试用例。观察官方的测试用例，官方向集合中插入了元素1，但是却试图删除集合中不存在的元素3，随后调用getRandom方法访问集合中的一个元素。由于这里我们采用了`unordered_map<int, int>`这个容器，因此当删除不存在的元素时，虽然remove方法可以正确返回false，但这仍然使得`unordered_map通过this->m[val]=0;`这个语句向`unordered_map<int, int>`中插入了新的元素3（虽然它对应的value仍为0）。这使得我们这里使用getRandom方法访问起始迭代器对应的元素时，会将并不“存在”的元素3遍历出来。因此这里我们需要加一层验证，检查迭代器指向的元素是否存在，若不存在则移动迭代器。

修改后的代码如下。注意下面的代码还为了保证每个元素都有可能被遍历到而加入了偏移机制。

```cpp
class RandomizedSet {
private:
    unordered_map<int, bool> m;
    int length;
    int offset;

public:
    RandomizedSet() {
        this->length = 0;
        this->offset = 0;
    }

    bool insert(int val) {
        if (this->m[val] == false) { // 查询该元素是否存在
            this->m[val] = true;
            this->length++;
            this->offset %= this->length;
            return true;
        } else
            return false;
    }

    bool remove(int val) {
        if (this->m[val] == true) { // 查询该元素是否存在
            this->m[val] = false;
            this->length--;
            if (this->length == 0)
                this->offset = 0;
            else
                this->offset %= this->length;
            return true;
        } else {
            this->m[val] = false;
            return false;
        }
    }

    int getRandom() {
        auto iter = (this->m).begin();
        for (int i = 0; i < this->offset; i++) {
            iter++;
            if (iter == (this->m).end())
                iter = (this->m).begin();
        }
        this->offset++;
        this->offset %= this->length;
        while ((*iter).second == false) {
            iter++;
            if (iter == (this->m).end())
                iter = (this->m).begin();
        }
        return (*iter).first;
    }
};
```

但遗憾的是，这个程序依然没有通过。在一次大量getRandom操作的测试中，上面的程序最终“解答错误”。具体原因不明，可能是输出的结果不够随机。ChatGPT分析问题说我的代码运行超时了，并且采用offset移位的方法也做不到真正的随机输出。

我苦思冥想半天想不出怎么做出真正的随机输出，最终无果，只能查看官方题解。没想到官方直接采用了rand函数实现了随机读写，没想到居然是大道至简的这一条路……

此外官方的做法也很巧妙，采用了“哈希表+数组”的方法，通过哈希表实现`$O(1)$`时间的查找，再通过数组实现随机读写——随机读写只需要在数组的所有索引中随机即可。下面是ChatGPT给出的官方解法：

```cpp
class RandomizedSet {
private:
    unordered_map<int, int> m;    // 存储元素的值和对应的索引
    vector<int> nums;             // 存储所有元素，支持 O(1) 随机访问

public:
    RandomizedSet() {}

    bool insert(int val) {
        if (m.find(val) != m.end()) {  // 如果元素已存在
            return false;
        }
        m[val] = nums.size();          // 存储元素和其索引
        nums.push_back(val);           // 在数组中添加元素
        return true;
    }

    bool remove(int val) {
        if (m.find(val) == m.end()) {  // 如果元素不存在
            return false;
        }

        int last = nums.back();        // 获取最后一个元素
        int idx = m[val];              // 获取要删除元素的索引

        nums[idx] = last;              // 将最后一个元素移到要删除元素的位置
        m[last] = idx;                 // 更新最后一个元素的索引

        nums.pop_back();               // 删除最后一个元素
        m.erase(val);                  // 删除元素值

        return true;
    }

    int getRandom() {
        int randomIndex = rand() % nums.size();  // 随机选取索引
        return nums[randomIndex];                // 返回随机元素
    }
};
```

回顾整个做题的过程，一个最重要的经验就是：**要实现`$O(1)$`时间的查找，必然要采用哈希表这种数据形式。**
