将一个给定字符串 `s` 根据给定的行数 `numRows` ，以从上往下、从左到右进行 Z 字形排列。

比如输入字符串为 `"PAYPALISHIRING"` 行数为 `3` 时，排列如下：

    P   A   H   N
    A P L S I I G
    Y   I   R

之后，你的输出需要从左往右逐行读取，产生出一个新的字符串，比如：`"PAHNAPLSIIGYIR"`。

请你实现这个将字符串进行指定行数变换的函数：

    string convert(string s, int numRows);

***

先观察一下官方题解给出的例子，关注输出的字符在原始一维字符串中的跳跃顺序。

    输入：s = "PAYPALISHIRING", numRows = 4
    输出："PINALSIGYAHRPI"
    解释：
    P     I    N
    A   L S  I G
    Y A   H R
    P     I

从左向右输出，先看第一行，首先输出了第0个字符，然后是第6个字符（+6），随后是第12个字符（+6）。

    P     I    N
    0     +6   +6

然后是第二行，首先输出了第1个字符，然后是第4个字符（+4），然后是第6个字符（+2），然后是第10个字符（+4），然后是第12个字符（+2）。

    A   L  S    I  G
    1   +4 +2   +4 +2

然后是第三行，首先输出了第2个字符，然后是第4个字符（+2），然后是第8个字符（+4），然后是……

    Y A    H  R
    2 +2   +4 +2

然后是最后一行，首先输出了第3个字符，然后是第9个字符（+6）。

    P     I
    3     +6

注意各行输出过程中输出元素索引的“增量”，注意这里我们我们呈现规律，使用了0来填空：

    第1行  初始0  +6 +0 +6 +0 ……
    第2行  初始1  +4 +2 +4 +2 ……
    第3行  初始2  +2 +4 +2 +4 ……
    第4行  初始3  +0 +6 +0 +6 …… 

观察行数取其他值时的例子，不难总结出整个输出的规律。如果总行数为`$n$`，那么一个“N”字形结构就由`$2(n-2)+2=2n-2$`个字符组成。当输出到第`$i$`行时，输出索引的起始值为`$i-1$`，随后按`$+2(n-i)$`，`$+2(i-1)$`……的增量增加。基于这个规律，我们便很容易写出程序：

```cpp
class Solution {
public:
    string convert(string s, int numRows) {
        if (numRows == 1)
            return s; // 特殊情况

        string ans = "";
        for (int i = 1; i <= numRows; i++) { // i为要输出的行数
            // 计算出两个实际增量
            int delta1 = 2 * (numRows - i);
            int delta2 = 2 * (i - 1);
            int delta = 1;

            // 在原始字符串中跳跃输出字符
            for (int j = i - 1; j < s.size(); j += delta) {
                if (delta != 0)
                    ans += s[j];
                if (delta == delta1) { // 在两个增量间来回切换
                    delta = delta2;
                } else {
                    delta = delta1;
                }
            }
        }

        return ans;
    }
};
```

官方题解给出了最直观的用矩阵存储上述“N”字形结构的方法。这种方法虽然直观，但是需要消耗额外空间来存储矩阵（二维数组），这样的时间和空间复杂度为`$O(rn)$`，其中`$r$`为行数，`$n$`为字符串的长度。这样做显然是不经济的，即使压缩矩阵的存储空间也并不经济。因此，不如我们上面提出的直接寻找规律构造的方法。官方答案中给出的结果更为简练：

```cpp
class Solution {
public:
    string convert(string s, int numRows) {
        int n = s.length(), r = numRows;
        if (r == 1 || r >= n) {
            return s;
        }
        string ans;
        int t = r * 2 - 2;
        for (int i = 0; i < r; ++i) { // 枚举矩阵的行
            for (int j = 0; j + i < n; j += t) { // 枚举每个周期的起始下标
                ans += s[j + i]; // 当前周期的第一个字符
                if (0 < i && i < r - 1 && j + t - i < n) {
                    ans += s[j + t - i]; // 当前周期的第二个字符
                }
            }
        }
        return ans;
    }
};
```

这种方法的时间复杂度为`$O(n)$`（只需要遍历一次字符串），空间复杂度为`$O(1)$`（只需要常数级的额外空间）。
