## 1. **结构类标签**
### `<div>`
- **作用**：块级容器，用于布局或包裹一段内容。
- **常用属性**：
- `id`：唯一标识
- `class`：用于选择器分组
- `style`：内联样式
- **示例**
```html
<div class="container" id="main" style="padding: 10px;">
  内容...
</div>
```
### `<span>`
- **作用**：行内元素，适合包裹小段文本或少量元素，不换行。
- **常用属性**：同 div
- **示例**
```html
<span style="color: red;">重点文字</span>
```
### `<h1>` ~ `<h6>`
- **作用**：标题元素，`h1` 最大，`h6` 最小。
- **属性**：
- `id`：可用于锚点跳转
- `class、style`
- **示例**
```html
<h1>一级标题</h1>
<h3 id="section1">第三层标题</h3>
```
## 2. **文本与内容类标签**
###  `<p>`
- **作用**：段落文本，块级元素。
```html
<p>这是一个段落。</p>
```
### `<br>`
- **作用**：换行（无闭合标签）。
```html
第一行<br>第二行
```
### `<strong>` / `<b>`
- **作用**：加粗，其中 `<strong>` 表示语义更强。
```html
<strong>重要信息</strong>
```
### `<em>` / `<i>`
- **作用**：倾斜，`<em>` 表语义强调。
```html
<em>重点内容</em>
```
## 3. **链接与媒体标签**
### `<a>`
- **作用**：超链接。
- **常用属性**：
- `href`：跳转地址
- `target="_blank"`：新窗口打开
- `rel="noopener noreferrer"`：安全属性
- **示例**
```html
<a href="https://example.com" target="_blank">访问链接</a>
```
### `<img>`
- **作用**：图片
- **常用属性**：
- `src`：图片路径
- `alt`：加载失败时的替代文本
- `width` / `height`
- **示例**
```html
<img src="cat.jpg" alt="一只猫" width="200">
```
### `<video>`
- **作用**：嵌入视频
- **常用属性**：
- `src`
- `controls`：显示控制栏
- `autoplay`
- `loop`
- `muted`
- **示例**
```html
<video src="movie.mp4" controls width="400"></video>
```
### `<audio>`
```html
<audio src="song.mp3" controls></audio>
```
## 4. **列表类**
### `<ul>` / `<ol>` / `<li>`
- **作用**：无序/有序列表
```html
<ul>
  <li>项目一</li>
  <li>项目二</li>
</ul>
```
## 5. **表格标签**
### 常见标签
- `<table>`：表格容器
- `<tr>`：表格行
- `<td>`：单元格
- `<th>`：表头
- `<thead>` `<tbody>` `<tfoot>`：三块区域
- `<caption>`：标题
### 常用属性
- `border`：边框
- `cellpadding` / `cellspacing`（老属性）
- `width`, `align`
### 示例
```html
<table border="1">
  <caption>示例表格</caption>
  <thead>
<tr>
  <th>姓名</th>
  <th>年龄</th>
</tr>
  </thead>
  <tbody>
<tr>
  <td>张三</td>
  <td>24</td>
</tr>
  </tbody>
</table>
```
## 6. **表单 Form**
### `<form>`
- **作用**：表单提交数据
- **常用属性**：
- `action`：提交地址
- `method="GET/POST"`
- `enctype="multipart/form-data"`：文件上传
### `<input>`
- **作用**：输入框
- **常用属性**：
- `type`：text, password, radio, checkbox, number, file, submit…
- `name`
- `placeholder`
- `value`
- **示例**
```html
<input type="text" name="username" placeholder="输入用户名">
```
### `<label>`
- **作用**：表单描述，可点击
```html
<label for="uid">用户名</label>
<input id="uid" type="text">
```
### `<textarea>`
- **多行输入框**
```html
<textarea rows="4" cols="30"></textarea>
```
### `<select>` `<option>`
```html
<select name="city">
  <option value="bj">北京</option>
  <option value="sh">上海</option>
</select>
```
### `<button>`
- 支持事件、submit、reset
```html
<button type="submit">提交</button>
```

## 7. **语义化标签（HTML5）**
- `<header>`：页头
- `<footer>`：页脚（常用作版权区域）
- `<nav>`：导航
- `<section>`：章节
- `<article>`：独立文章
- `<aside>`：侧栏
- `<main>`：主体内容
示例：
```html
<article>
  <header>文章标题</header>
  <p>内容……</p>
</article>
```
## 8. **容器/布局辅助标签**
### `<iframe>`
用于嵌入外部页面
```html
<iframe src="page.html" width="600" height="300"></iframe>
```
### `<hr>`
水平分割线
```html
<hr>
```
